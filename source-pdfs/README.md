# 題目原始 PDF 放置區

請把台灣內科醫學會的**題目**與**答案** PDF 放在這個資料夾。

## 怎麼放

1. 在手機 / 電腦開啟 github.com 進入本 repo
2. 進入 `source-pdfs/` 資料夾 → **Add file** → **Upload files**
3. 選擇 PDF 上傳 → **Commit changes**（commit 到 `claude/taiwan-med-exam-app-planning-WZgcu` 分支）

## 建議的命名方式

方便後續轉檔對應，建議用「年份 + 類型」命名，例如：

```
source-pdfs/
├── 2023-questions.pdf   # 2023 題目
├── 2023-answers.pdf     # 2023 答案
├── 2022-questions.pdf
└── 2022-answers.pdf
```

（命名不強制，能對得起來即可；上傳後再告訴我哪份是題目、哪份是答案也行。）

## 提醒

- 本 repo 為私有、不公開。
- 放入前請留意醫學會題庫的使用條款是否限制再散布。
- 第一步（Phase 0）只需要**一年份**的題目 + 答案，先跑通流程即可，不必一次全上傳。
