# 題目科目分類說明

本文件記錄題庫的科目分類規律、各年題號分佈,以及需人工判斷的「跨科/邊界題」清單,方便日後查核與修正。

> 分類規則寫在 [`tools/classify.py`](../tools/classify.py),如需修改某題科目,直接改該檔的 `OVERRIDES`/區塊範圍後執行 `python3 tools/classify.py` 重跑即可。

---

## 一、排列規律(觀察 113、114 兩年)

1. **非固定題數**:同一科目逐年題數會差一兩題(例:心臟血管 113 年 19 題、114 年 18 題),因此沒有「固定第幾題到第幾題」的死規則,需逐年對照。
2. **按科目連續成塊**排列,且分**兩堂**:第一堂約 Q1–80、第二堂約 Q81–160(與圖檔標示的「第一堂/第二堂」一致)。
3. **大致順序**:
   - **第一堂**:心臟血管 → 胸腔(含重症 / ARDS / 呼吸衰竭)→ 消化 → 腎臟 → **Q73–80 跨科整合題**
   - **第二堂**:感染 → 新陳代謝與內分泌 → 風濕免疫及過敏 → 血液腫瘤 → 神經 / 精神 → 皮膚
4. **Q73–80 為「跨科整合題」**(兩年皆然,每題各屬不同科);卷末**神經 / 精神**亦有交錯。這些題目以逐題方式依內容判定。

### 其他邊界題

- **114 Q21**(神經阻斷性惡性症候群 NMS):位於胸腔區塊中,依內容歸 **精神**。
- **113 Q75**(糖尿病+CKD 第四期+代謝性酸中毒):歸 **新陳代謝與內分泌**(亦可視為腎臟)。
- **113 Q77**(僵直性脊椎炎合併主動脈閉鎖不全):題目問主動脈瓣理學檢查,歸 **心臟血管**(底病為風濕免疫)。
- **113 Q113**(AZ 疫苗後 VITT,anti-PF4 抗體):位於風濕免疫區塊開頭,歸 **風濕免疫及過敏**(亦涉血栓/血液)。
- **卷末神經 / 精神交錯(Q145–155)**:中風、重症肌無力、Wernicke 腦病變、抗凝相關腦出血 → **神經**;NMS、血清素症候群、憂鬱症、失智 BPSD、失眠 → **精神**。

### 其他可注意的跨科主題

- **結核(TB)**:可同時屬胸腔與感染。本分類中,胸腔區塊內的 TB 歸胸腔;整合題/篩檢類(如 114 Q77)歸感染。
- **重症(ARDS、敗血症、呼吸衰竭)**:歸於胸腔(含重症)。
- **各科癌症**(胃癌、胰臟癌、大腸癌等):若位於消化區塊則歸消化;位於血液腫瘤區塊則歸血液腫瘤。
