### 105-180 ・ 正解 C ・ status: draft

**題幹**  
60歲男性因主動脈剝離住院，住院中發生四肢無力與失禁，神經學檢查顯示四肢肌力為1分，四肢肌腱反射增加，蹠反射（plantar reflex）為向上，針刺感覺（pin prick sense）自胸部以下喪失，關節活動感覺（joint position sense）則正常，何種病變的可能性最高？

- (A) 硬腦膜外膿瘍（epidural abscess）
- (B) 硬腦膜下血腫（subdural hematoma）
- (C) 前脊椎動脈症候群（anterior spinal artery syndrome） ← 正解
- (D) 急性橫貫性脊髓炎（acute transverse myelitis）
- (E) 硬腦膜外血腫（epidural hematoma）

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**脊髓血管症候群的定位診斷**:在主動脈剝離(aortic dissection)病人,剝離可阻斷供應脊髓前2/3的**Adamkiewicz 動脈/前脊椎動脈**,造成**前脊椎動脈症候群(anterior spinal artery syndrome, ASAS)**。關鍵定位線索在「**分離性感覺缺失**」:前索/前外側索(脊髓視丘徑)受損 → **針刺(痛溫覺)自病灶以下喪失**;後索(dorsal column)由後脊椎動脈供應而保留 → **關節位置覺/振動覺正常**。本例四肢肌力1分、腱反射亢進、Babinski 陽性(上運動神經元徵象)、合併膀胱失禁,加上痛溫覺喪失而本體覺保留,完全符合 ASAS。

### 選項分析
- **(A) 硬腦膜外膿瘍(epidural abscess)** — 錯誤。為亞急性發燒+背痛+局部壓痛+進行性脊髓壓迫,通常**所有感覺型態(含本體覺)皆受影響**,且與主動脈剝離無因果。本題無感染病程,且為「分離性感覺缺失」。
- **(B) 硬腦膜下血腫(subdural hematoma)** — 錯誤。為**顱內**病灶,造成偏癱與意識障礙,不會產生胸髓平面以下對稱性截癱與感覺平面;此為「腦/脊」混淆陷阱。
- **(C) 前脊椎動脈症候群(anterior spinal artery syndrome)** — **正確 → 本題答案**。主動脈剝離阻斷脊髓前血流,前2/3梗塞 → 雙側運動麻痺+痛溫覺喪失(脊髓視丘徑),後索保留 → **本體覺正常**,即典型「本體覺豁免」的分離性感覺缺失。
- **(D) 急性橫貫性脊髓炎(acute transverse myelitis)** — 錯誤。為發炎性、**橫斷整條脊髓**,通常**所有感覺型態(含本體覺)皆缺損**,且為亞急性、與發炎/感染後相關;本題本體覺保留、且有主動脈剝離的血管病因,不符。
- **(E) 硬腦膜外血腫(epidural hematoma)** — 錯誤。脊髓硬膜外血腫多急性背痛起病、機械性壓迫,亦會影響後索功能;與主動脈剝離的血管梗塞機轉不同,且非「分離性感覺缺失」。

### 答案解析
題問「可能性最高」。主動脈剝離 + 急性截癱 + 痛溫覺喪失但**本體覺保留**(分離性感覺缺失)= 前脊椎動脈梗塞,故選 **(C)**。

### 核心知識點
- **ASAS 三聯**:病灶以下**雙側無力**(皮質脊髓徑)+ **痛溫覺喪失**(脊髓視丘徑)+ **本體覺/振動覺保留**(後索豁免);常伴急性尿失禁、初期脊髓休克可呈鬆弛無反射,數日後轉為亢進+Babinski。
- **血管解剖**:**1條前脊椎動脈供應前2/3**、2條後脊椎動脈供應後1/3;胸髓 **T4–T8** 為分水嶺最易缺血;**Adamkiewicz 動脈**多源自 T9–T12 左側。
- **病因**:主動脈剝離/手術、嚴重低血壓、栓塞、纖維軟骨栓塞;主動脈剝離以脊髓梗塞為初現者罕見(約4%)。
- **對比**:後索受損(如B12缺乏、tabes)→ 振動/位置覺喪失而痛溫覺保留,與 ASAS 相反。

### 105-181 ・ 正解 B ・ status: draft

**題幹**  
20歲男性，因睡眠問題看診，主訴有下列症狀，何者是診斷猝睡症（narcolepsy）最明確（specific）的症狀？

- (A) 白天上課時很容易睡著
- (B) 大笑時會全身無力 ← 正解
- (C) 常一躺下就睡著了
- (D) 白天精神常不佳
- (E) 剛睡醒時有幻覺

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**猝睡症(narcolepsy)四聯症中最具特異性(specific)的症狀**。猝睡症四聯症:①日間過度嗜睡(EDS,最敏感、cardinal,但不特異)、②**猝倒(cataplexy)**、③睡眠麻痺(sleep paralysis)、④入睡/醒前幻覺(hypnagogic/hypnopompic hallucinations)。其中**猝倒**——由**情緒(尤其大笑)誘發、意識清醒下的雙側肌肉張力突然喪失**——與**下視丘 hypocretin/orexin 神經元缺失(narcolepsy type 1)**高度連結,是診斷最具特異性的線索。本題「大笑時全身無力」即典型猝倒。

### 選項分析
- **(A) 白天上課時很容易睡著** — 錯誤(非最特異)。即日間過度嗜睡(EDS),是猝睡症最常見、最敏感的症狀,但**睡眠不足、阻塞性睡眠呼吸中止、憂鬱**等都會出現,特異性低。
- **(B) 大笑時會全身無力** — **正確 → 本題答案**。即**猝倒(cataplexy)**:情緒(大笑、驚訝)誘發、清醒下雙側肌張力驟失,反映 REM 肌張力抑制機制侵入清醒期;與 hypocretin 缺失強相關,**對 narcolepsy type 1 特異性最高**。
- **(C) 常一躺下就睡著了** — 錯誤。即入睡潛伏期極短,屬 EDS/睡眠剝奪表現,非特異;陷阱在與「sleep-onset REM」混淆,但單純快速入睡並不專屬猝睡症。
- **(D) 白天精神常不佳** — 錯誤。非特異的主觀疲倦,可見於任何睡眠不足或情緒障礙。
- **(E) 剛睡醒時有幻覺** — 錯誤。即醒前幻覺(hypnopompic),屬 REM 侵入現象,雖支持診斷,但**一般人睡眠剝奪亦可出現**,特異性不如猝倒。

### 答案解析
題問「最明確(specific)的症狀」。EDS 最敏感但不特異;唯有**猝倒**對猝睡症(type 1)特異性最高,故選 **(B)**。

### 核心知識點
- **Narcolepsy type 1(NT1)**:EDS + **猝倒** 或 **CSF hypocretin-1 ≤110 pg/mL**;type 2(NT2):有 EDS、MSLT 陽性但**無猝倒、hypocretin 正常**。
- **診斷工具**:夜間多項睡眠檢查後接 **MSLT**——平均入睡潛伏期 **≤8 分鐘 + ≥2 次 sleep-onset REM periods(SOREMP)**(夜間 PSG 出現的 SOREMP 可計1次)。
- **猝倒特徵**:意識保留、由情緒誘發、雙側、可局部(臉部抽動、下巴鬆弛、膝軟),與癲癇、暈厥鑑別。
- **HLA**:NT1 與 **HLA-DQB1*06:02** 強相關,支持自體免疫破壞 hypocretin 神經元假說。

### 參考資料
1. Scammell TE. Narcolepsy. *N Engl J Med*. 2015;373(27):2654-2662. PMID: [26716917](https://pubmed.ncbi.nlm.nih.gov/26716917/). DOI: [10.1056/NEJMra1500587](https://doi.org/10.1056/NEJMra1500587).
2. American Academy of Sleep Medicine. *International Classification of Sleep Disorders (ICSD-3-TR)*. 3rd ed., text revision. AASM; 2023（Narcolepsy type 1 / type 2 diagnostic criteria）.（現行版）

</details>

### 105-182 ・ 正解 E ・ status: draft

**題幹**  
40歲女性因大量流汗、發燒、意識不清被送至急診，檢查顯示脈搏每分鐘120下、肢體僵直(rigidity)、肌腱反射增加 (hyperreflexia)。病人有憂鬱症病史，在精神科門診看診用藥，最有可能是下列何診斷？

- (A) 韋尼克氏腦病（Wernicke's encephalopathy）
- (B) 病毒腦炎（viral encephalitis）
- (C) 抗精神藥物惡性症候群（neuroleptic malignant syndrome）
- (D) 鎮靜劑過量（tranquilizer overdose）
- (E) 血清症候群（serotonin syndrome） ← 正解

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**高熱+自主神經不穩+神經肌肉異常的鑑別**,核心在區分**血清素症候群(serotonin syndrome, SS)**與**抗精神病藥惡性症候群(neuroleptic malignant syndrome, NMS)**。鑑別關鍵在**神經肌肉表現**:本例「肌腱反射增加(hyperreflexia)」加上肢體僵直、高熱、心跳120、意識改變,**反射亢進(±陣攣)指向 SS**;NMS 的特徵反而是「**鉛管樣僵直 + 反射低下/正常**」、起病緩慢。憂鬱症病史+精神科用藥(極可能含 SSRI/SNRI)更支持 SS。

### 選項分析
- **(A) 韋尼克氏腦病(Wernicke's encephalopathy)** — 錯誤。為硫胺(B1)缺乏所致,三聯為**眼肌麻痺+步態失調+意識混亂**,不以高熱、僵直、反射亢進為特徵。
- **(B) 病毒腦炎(viral encephalitis)** — 錯誤。可發燒+意識改變,但缺乏「用藥誘發+對稱反射亢進/陣攣」的毒性症候群型態;需 CSF/MRI 佐證,非本例最佳解釋。
- **(C) 抗精神病藥惡性症候群(NMS)** — 錯誤(為干擾項)。NMS 由**多巴胺 D2 阻斷**引起,典型為**鉛管樣僵直 + 反射低下 + 起病數日**、CK 大幅升高;本例「**反射亢進**」與快速起病不符 NMS。
- **(D) 鎮靜劑過量(tranquilizer overdose)** — 錯誤。BZD/鎮靜劑過量表現為**鎮靜、呼吸抑制、反射低下、低體溫**,與本例高熱、反射亢進相反。
- **(E) 血清症候群(serotonin syndrome)** — **正確 → 本題答案**。血清素過度活化 → **神經肌肉亢進(陣攣、反射亢進、僵直)+ 自主神經不穩(高熱、心搏過速、出汗)+ 精神狀態改變**;憂鬱症用藥(SSRI/SNRI)為典型誘因,起病多在用藥/加量後24小時內。

### 答案解析
題問「最有可能」。憂鬱症服藥 + 高熱、心搏過速、僵直且**反射亢進**(而非低下)= 血清素症候群,故選 **(E)**。

### 核心知識點
- **SS vs NMS 速記**:**反射亢進+陣攣+快速起病 → SS**;**鉛管樣僵直+反射低下+緩慢起病(數日) → NMS**。
- **Hunter 準則(SS)**:用血清素藥下,符合任一:自發性陣攣 / 誘發性陣攣+躁動或出汗 / 眼陣攣+躁動或出汗 / 顫抖+反射亢進 / 高張+體溫>38°C+眼或誘發性陣攣;敏感度84%、特異度97%。
- **處置**:停藥+支持治療+降溫;SS 嚴重者用 **cyproheptadine 12 mg PO 起,續 2 mg Q2h(最多 32 mg/24h)**;NMS 用 **dantrolene、bromocriptine**;兩者重症皆可能需鎮靜麻痺、插管。
- **誘因藥**:SS——SSRI/SNRI、MAOI、tramadol、linezolid、triptan、止吐藥(metoclopramide)、dextromethorphan、哌替啶(meperidine);NMS——典型/非典型抗精神病藥、止吐 D2 阻斷劑。

### 105-183 ・ 正解 A ・ status: draft

**題幹**  
30歲女性因雙下肢無力、麻木與失禁就醫，脊髓液檢查顯示無白血球，葡萄糖為70 mg/dL、全部蛋白為75 mg/dL、白蛋白為50 mg/dL、IgG為20 mg/dL，血清的白蛋白為 3000 mg/dL、IgG為1000 mg/dL，則她的IgG index是多少？

- (A) ≧1.20 ← 正解
- (B) 1.00~1.19
- (C) 0.80~0.99
- (D) 0.60~0.79
- (E) <0.60

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**IgG index 的計算與判讀**,用於評估**中樞神經內鞘內 IgG 合成(intrathecal IgG synthesis)**,是多發性硬化症(MS)等發炎性脫髓鞘的重要指標。公式以白蛋白(僅由血液漏入 CNS)為「血腦屏障通透」的內部對照,校正血源性 IgG:

**IgG index = (CSF IgG / 血清 IgG) ÷ (CSF 白蛋白 / 血清白蛋白)**。

代入本題:CSF IgG 20、血清 IgG 1000、CSF 白蛋白 50、血清白蛋白 3000：
(20/1000) ÷ (50/3000) = 0.02 ÷ 0.01667 = **1.20** → 落在「≧1.20」。

### 選項分析
- **(A) ≧1.20** — **正確 → 本題答案**。計算結果恰為 1.20,且 >0.7 表示**內鞘內 IgG 合成增加**,支持 MS/慢性 CNS 發炎。
- **(B) 1.00~1.19** — 錯誤。漏算或四捨五入錯誤的陷阱(若把比值算成約1.0)。
- **(C) 0.80~0.99** — 錯誤。常見於把分子分母顛倒或單位混用所致的誤算。
- **(D) 0.60~0.79** — 錯誤。此區間接近正常上限(<0.7 正常),與本題數據不符。
- **(E) <0.60** — 錯誤。屬正常範圍,代表無內鞘合成,與本例計算不符。

### 答案解析
題問「IgG index 是多少」。依公式計得 1.20,故選 **(A)**。

### 核心知識點
- **公式**:IgG index = (CSF IgG/血清 IgG) ÷ (CSF Alb/血清 Alb);**正常 ≤0.7**(部分實驗室 0.85),**>0.7 提示內鞘內 IgG 合成**,約見於75% MS。
- **白蛋白商(Q-Alb = CSF Alb/血清 Alb)**:評估**血腦屏障完整性**;本例 Q-Alb = 50/3000 ≈ 0.017(正常),提示蛋白升高(75 mg/dL)主要來自內鞘合成而非屏障破壞。
- **搭配 oligoclonal bands(OCB)**:CSF 出現而血清無的條帶,診斷敏感度高於 IgG index;兩者並用評估 MS。
- **臨床情境**:本例30歲女性、雙下肢無力麻木失禁、CSF 無白血球但 IgG index 升高 → 高度懷疑脫髓鞘性脊髓病變(如 MS/NMOSD),需 MRI 與 OCB、AQP4 抗體進一步釐清。

### 本題觀念
本題考**職業性重金屬中毒導致的續發性巴金森氏症(secondary parkinsonism)**。關鍵線索:①工廠職業暴露、②**同廠多名同事出現類似症狀**(指向共同環境毒物)、③漸進性運動遲緩、步幅變小、平衡變差。**錳(manganese)中毒(manganism)**經呼吸道吸入後蓄積於**基底核(尤其蒼白球 globus pallidus)**,造成巴金森樣症候群,是焊接、採礦、合金、電池等行業的典型職業病。

### 105-185 ・ 正解 D ・ status: draft

**題幹**  
慢性腎臟疾病（chronic kidney disease）病人接受透析治療的時候，發生透析失衡症候群（dialysis disequilibrium syndrome），下列敘述何者為非？

- (A) 多在透析結束時發生，而非透析開始前發生
- (B) 會表現突眼、眼壓高
- (C) 會表現頭痛、癲癇
- (D) 因水分自腦部脫出太多所致 ← 正解
- (E) 因移除尿素太快所致

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**透析失衡症候群(dialysis disequilibrium syndrome, DDS)的機轉**——這是一題「何者為非」題,務必選**錯誤敘述**。DDS 好發於**初次透析或重度氮血症(高 BUN)病人**,因血中尿素被快速移除,而尿素跨越血腦屏障速度慢,造成**血漿與腦組織間短暫滲透壓梯度,水分由血漿移入腦細胞 → 腦水腫**(「逆滲透轉移 reverse osmotic shift」)。臨床表現為頭痛、噁心、抽搐、意識改變,甚至顱內/眼內壓上升。

### 選項分析
- **(A) 多在透析結束時發生,而非透析開始前** — 正確敘述。DDS 多於透析**進行中或結束時/結束後不久**出現,符合尿素快速下降造成滲透梯度的時序。
- **(B) 會表現突眼、眼壓高** — 正確敘述。腦水腫伴隨**顱內壓與眼內壓上升**,故可有頭痛、眼壓高等表現。
- **(C) 會表現頭痛、癲癇** — 正確敘述。腦水腫引發頭痛、噁心、躁動、抽搐(癲癇)、嚴重者昏迷,為典型神經症狀。
- **(D) 因水分自腦部脫出太多所致** — **為非(錯誤敘述)→ 本題答案**。機轉恰相反:是**水分移入腦部造成腦水腫**,而非「水分自腦部脫出」。此即經典反向陷阱。
- **(E) 因移除尿素太快所致** — 正確敘述。**快速清除尿素**造成血漿滲透壓驟降、血腦間滲透梯度,正是 DDS 核心成因。

### 答案解析
題問「何者為非」。DDS 是水分**進入**腦部造成腦水腫,選項(D)說成「水分自腦部脫出」屬錯誤,故答案為 **(D)**。

### 核心知識點
- **機轉**:重度氮血症初次透析 → 血尿素快降、腦內尿素清除慢 → **腦-血滲透梯度 → 水分入腦 → 腦水腫、顱內壓↑**;另有「腦內 pH 下降/特異滲透物(idiogenic osmoles)」假說。
- **高危**:**首次透析、BUN 顯著升高(如 >175 mg/dL)、老人/幼兒、既有 CNS 病變、嚴重代謝性酸中毒**。
- **預防**:首次透析採**低效率、短時間、低血流量**,逐步增加;必要時加 mannitol、提高透析液鈉,或改用緩和的 CRRT/PD。
- **鑑別**:與低血糖、低/高血鈉、硬膜下血腫、尿毒性腦病、空氣栓塞等透析併發症區分。

### 105-186 ・ 正解 B ・ status: draft

**題幹**  
25歲男性1年前曾發生車禍致鎖骨骨折，半年前起漸進右上肢無力、麻木與冰冷就診，神經學檢查顯示右上肢內側感覺遲鈍、右手肌力較差，右臂血壓為90/60 mmHg、左臂血壓為130/80 mmHg，最可能診斷為何？

- (A) 頸神經根病變（cervical radiculopathy）
- (B) 胸廓出口症候群（thoracic outlet syndrome) ← 正解
- (C) 臂神經叢病變（brachial plexopathy）
- (D) 腕隧道症候群（carpal tunnel syndrome）
- (E) 鎖骨下動脈竊血症候群（subclavian steal syndrome） < 一○五年度試題目錄 >

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**鎖骨骨折後的胸廓出口症候群(thoracic outlet syndrome, TOS),尤其動脈型(arterial TOS)**。定位線索:①1年前鎖骨骨折病史(骨痂/畸形癒合壓迫鎖骨下血管神經束)、②進行性右上肢無力、麻木、**冰冷**(缺血)、③**右臂血壓 90/60 明顯低於左臂 130/80(雙臂收縮壓差 >20 mmHg)**——強烈指向**鎖骨下動脈受壓/狹窄**。神經(臂神經叢內側索/下幹)+動脈同時受累,符合 TOS 在鎖骨-第一肋間隙的壓迫。

### 選項分析
- **(A) 頸神經根病變(cervical radiculopathy)** — 錯誤。為單一頸神經根(如 C6/C7)受壓,呈皮節性疼痛/無力,**不會造成雙臂血壓差與肢體冰冷**(無動脈受累)。
- **(B) 胸廓出口症候群(thoracic outlet syndrome)** — **正確 → 本題答案**。鎖骨骨折後骨痂/畸形於胸廓出口壓迫**鎖骨下動脈(→血壓差、缺血、冰冷)與臂神經叢下幹(→內側感覺遲鈍、手肌無力)**,完美對應本例。
- **(C) 臂神經叢病變(brachial plexopathy)** — 錯誤(干擾項)。可解釋感覺/運動症狀,但**單純神經叢病變不會造成雙臂血壓差與肢體冰冷**;本例的動脈受累正是 TOS 而非單純叢病變。
- **(D) 腕隧道症候群(carpal tunnel syndrome)** — 錯誤。為正中神經於腕部受壓,症狀侷限**手掌橈側三指半**、夜間麻;與近端內側感覺、血壓差、缺血無關。
- **(E) 鎖骨下動脈竊血症候群(subclavian steal syndrome)** — 錯誤。為鎖骨下動脈**近端(椎動脈起點之前)**狹窄,血流由椎動脈倒灌,典型造成**後循環症狀(暈眩、復視)**且多在活動上肢時誘發;雖也有雙臂血壓差,但本例為鎖骨骨折局部壓迫、且以上肢神經缺血症狀為主,定位在胸廓出口而非近端鎖骨下動脈。

### 答案解析
題問「最可能診斷」。鎖骨骨折病史 + 上肢神經症狀 + 缺血(冰冷)+ **雙臂血壓差**,定位於胸廓出口的血管神經壓迫,故選 **(B)**。

### 核心知識點
- **TOS 三型**:**神經型(>90%)**(臂神經叢下幹 C8–T1,手內肌無力/內側麻木)、**靜脈型**(Paget-Schroetter,上肢腫脹發紺)、**動脈型**(鎖骨下動脈受壓→缺血、血壓差、栓塞)。
- **誘因**:頸肋、第一肋/鎖骨骨折畸形癒合、纖維帶、斜角肌肥厚。
