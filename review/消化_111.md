- (D) (3) + (5) + (7)。 ← 正解
- (E) (4) + (5) + (6)。

### 本題觀念
本題以複選考**肝生檢(liver biopsy)的角色與限制**:何時是首選、何時非首選、能做什麼(纖維化分級、不明原因肝炎、AIH 確診),以及禁忌(大量腹水時不可貿然經皮穿刺)。核心觀念:肝生檢仍是**評估肝纖維化與不明病因肝炎的金標準/重要工具**,但**膽汁滯留與血管性疾病(Budd-Chiari)的首選是影像而非切片**。

### 選項分析
- **(1) Cholestasis 首選肝生檢** — 錯誤。膽汁滯留首先要分**肝外阻塞 vs 肝內**,首選是**影像(超音波→MRCP/EUS)**找膽道擴張/阻塞,非一開始就切片。
- **(2) Budd-Chiari 首選肝生檢** — 錯誤。Budd-Chiari(肝靜脈流出阻塞)首選是**都卜勒超音波/CT/MR venography**評估肝靜脈與下腔靜脈血流,而非肝生檢。
- **(3) 肝生檢有助纖維化分級** — **正確(屬答案組)**。組織學分期(如 METAVIR)仍是纖維化評估的參考標準。
- **(4) non-invasive tests 已可精準取代肝生檢** — 錯誤。FIB-4、FibroScan(LSM)、ELF 等可**減少**切片需求,但在灰色地帶/病因不明時**尚不能完全取代**,「已可取代」過度宣稱。
- **(5) 有助不明原因肝炎之診斷** — **正確(屬答案組)**。病因不明的肝功能異常,組織學常提供關鍵線索。
- **(6) 明顯腹水疑肝內瀰漫腫瘤「祇能」經皮肝生檢** — 錯誤。大量腹水是**經皮穿刺相對禁忌(出血風險高)**,應採**經頸靜脈(transjugular)路徑**;「祇能經皮」錯誤。
- **(7) 有助 AIH 之確診** — **正確(屬答案組)**。AIH 診斷需組織學(介面性肝炎、漿細胞浸潤)配合抗體與 IgG。
- 故 **(3)+(5)+(7)** 為正確敘述組合 → **(D) → 本題答案**。其餘組合都摻入 (1)(2)(4)(6) 這些錯誤項。

### 答案解析
題問「哪些**正確**」。正確者為 (3) 纖維化分級、(5) 不明原因肝炎、(7) AIH 確診,即 **(D) (3)+(5)+(7)**;(1)(2) 首選為影像、(4) 非侵襲檢查尚不能全面取代、(6) 大量腹水應走經頸靜脈,皆錯。

### 核心知識點
- **肝生檢路徑**:一般經皮;**凝血異常、大量腹水→ 經頸靜脈(transjugular)**;局部病灶→ 影像導引。
- **非侵襲纖維化評估**:**FIB-4、APRI(血清)**;**FibroScan/MRE(影像彈性)**;用於分流,高風險或病因不明再切片。
- **首選影像而非切片的情境**:膽汁滯留(US→MRCP)、Budd-Chiari(都卜勒/venography)、局部腫瘤(動態影像)。
- **切片的不可取代價值**:**不明病因肝炎、AIH/PBC 重疊、藥物性肝損、纖維化精準分期**。

### 本題觀念
本題考**台灣健保慢性 B 肝口服抗病毒藥物給付規範**(以考年 111 之健保標準判定)。重點在各族群的**治療終點與療程**:HBeAg(+)、HBeAg(−)、肝硬化、肝癌、化療前預防性用藥。需挑出**錯誤**的一項。

### 選項分析
- **(A) HBeAg(+) 治療至 HBeAg 消失後再續服一年** — 正確。台灣健保 HBeAg(+) 之療程設計即以 **HBeAg seroconversion 後鞏固治療(consolidation)約一年**為停藥條件。
- **(B) HBeAg(−) 治療至 HBV DNA 連續三次(每半年)測不到,療程至少兩年、至多三年** — 正確。符合考年健保對 HBeAg(−) 之療程上限與停藥標準。
- **(C) 肝硬化只要驗得到病毒得長期服用** — 正確。**肝硬化(代償或失代償)病人健保放寬為長期/不限療程治療**,降低肝病進展與肝癌風險。
- **(D) 肝細胞癌病人只要驗得到病毒得「終生」服用** — **錯誤 → 本題答案**。健保對肝癌合併 B 肝雖放寬給付以抑制病毒、降低復發,但條文用語/給付架構並非「終生」無條件;此句以「終生」絕對化超出考年給付規範文字,故為錯誤敘述。臨床上肝癌病人多需長期抗病毒,但本題考的是**健保規範文字**。
- **(E) 化療前一週開始預防性用藥,至化療結束再服半年** — 正確。為防 **HBV reactivation**,化療/免疫抑制前即啟動預防,並於療程結束後**續用約 6–12 個月**,符合健保預防性給付精神。

### 答案解析
題問「何者**錯誤**」。依考年(111)健保給付規範,肝癌病人之給付並非如 (D) 所述「只要驗得到病毒得**終生**服用」,故 **(D)** 為錯誤、即本題答案。
> 健保給付條文逐年微調,實際以**台灣現行健保規範為準**;本題正解依 111 年標準。

### 核心知識點
- **HBeAg(+)**:治療至 **HBeAg seroconversion + 鞏固約 1 年**。
- **HBeAg(−)**:治療至 **HBV DNA 連續三次(每半年)測不到**,療程 2–3 年(考年規範)。
- **肝硬化**:可驗得病毒即**長期治療**。
- **預防 reactivation**:化療/免疫抑制/anti-CD20 前先給藥,療程後續用 **6–12 個月**(高風險者更久)。
- **首選藥**:**entecavir、tenofovir(TDF/TAF)**——高效、低抗藥性。

### 本題觀念
本題以複選考**台灣 C 肝(HCV)DAA 治療之健保給付與臨床原則**(以考年 111 標準)。涵蓋:DAA 為主流、是否需等待、開立資格、失代償肝硬化用藥限制、根除率、肝癌合併 C 肝的處置順序、HBV reactivation。需挑出**錯誤**敘述組合。

### 選項分析
- **(1) 以全口服 DAA 為治療主流** — 正確。台灣早已全面以 DAA 取代干擾素為主流。
- **(2) C 肝病人要等 6 個月才能接受 DAA** — **錯誤**。確診現症(HCV RNA 陽性)即可治療,**不需等待 6 個月**;早治療早阻斷傳播與肝病進展。
- **(3) 現在台灣執業西醫師皆可開立 DAA** — 正確。為加速消除 C 肝,**已放寬全西醫師**(不限肝膽腸胃/感染專科)可開立。

- **(5) 絕大多數 C 肝(含代償性肝硬化)DAA 根除率可達 98%** — 正確。現代 DAA **SVR12 >95–98%**,代償肝硬化亦然。

- **(7) B 肝/C 肝共感染者於 DAA 治療中 B 肝病毒可能被激活** — 正確。清除 HCV 後 **HBV reactivation** 風險上升(FDA 黑框),治療前須驗 HBsAg/anti-HBc。
- 錯誤者為 (2)、(4)、(6) → 組合 **(B) (2)+(4)+(6) → 本題答案**。

### 答案解析
題問「何者**錯誤**」。(2) 不需等 6 個月、(4) 失代償肝硬化反而**禁用** protease inhibitor(Maviret)、(6) 有肝癌應先治癌,三者錯誤,即 **(B)**。

### 核心知識點
- **治療前評估**:HCV RNA 確認現症、纖維化分期、**HBsAg/anti-HBc(防 reactivation)**、藥物交互作用、肝代償狀態。
- **代償(Child-Pugh A)**:可用含 PI 之 **glecaprevir/pibrentasvir** 或 **sofosbuvir/velpatasvir**。
- **失代償(Child-Pugh B/C)**:**禁用 protease inhibitor**;用 **sofosbuvir/velpatasvir ± ribavirin** 12–24 週。
- **HCC 合併 HCV**:先處置活動性腫瘤,再依狀況抗病毒;DAA 不延誤腫瘤治療。
- **政策**:全西醫師可開立,推動**消除 C 肝**目標。

### 本題觀念
本題考**膽囊疾病處置**:急性膽囊炎手術時機、無症狀膽結石(含糖尿病人)是否預防性切除、UDCA 溶石的適應與成效、無症狀膽結石的自然史、膽囊息肉切除門檻。需挑出**錯誤**敘述,關鍵在 UDCA 溶石的真實成效被誇大。

### 選項分析
- **(A) 無併發症急性膽囊炎,診斷後 48–72 小時內手術** — 正確。實證支持**早期腹腔鏡膽囊切除(發病/入院 72 小時內)**,併發症與住院天數較少。
- **(B) 糖尿病人無症狀膽結石不必預防性切除** — 正確。無症狀膽結石(即使糖尿病)**不建議常規預防性手術**;每年僅約 1–2% 轉為有症狀。
- **(C) radiolucent cholesterol stones 不論大小皆有 50% 機會以 UDCA 完全溶化** — **錯誤 → 本題答案**。UDCA 口服溶石**僅適用小(<5–10 mm)、無鈣化、純膽固醇結石且膽囊功能正常者**;「**不論大小**皆 50% 完全溶化」過度誇大——大結石/鈣化結石幾乎無效,且停藥後**易復發**。
- **(D) 意外發現之無症狀膽結石長期追蹤 60–80% 不產生症狀** — 正確。無症狀膽結石多數**終生不發病**,符合自然史數據。
- **(E) ≥50 歲且膽囊息肉 >1 cm 建議膽囊切除** — 正確。**息肉 ≥10 mm、年齡 >50、快速增大、合併膽結石/PSC** 屬惡性風險因子,建議切除。

### 答案解析

### 111-054 ・ 正解 E ・ status: draft

**題幹**  
40歲女性在飯後30至60分鐘後，有上腹痛、腹脹、噁心和腹瀉，已發生三個月，有時大餐後合併頭痛、流汗現象，上述症狀在飯後30分鐘至2小時會緩解。本身為肥胖、高血壓和第二型糖尿病患者，四個月前接受減重手術sleeve gastrectomy。胃鏡檢查除sleeve gastrectomy外並無異常，胃切片無幽門螺旋桿菌，十二指腸切片及胃排空檢查均為正常。此位患者最恰當的治療為?

- (A) acarbose。
- (B) metformin。
- (C) metoclopramide。
- (D) rifaximin。
- (E) 少量多餐。 ← 正解

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題為**減重手術(sleeve gastrectomy)後之傾食症候群(dumping syndrome)**:40 歲、術後 4 個月、**飯後 30–60 分鐘上腹痛、腹脹、噁心、腹瀉**,大餐後合併**頭痛、流汗(自律神經/血管運動症狀)**,症狀於餐後 30 分鐘–2 小時緩解;胃鏡(除 sleeve 外)、切片、胃排空均正常。考點為 dumping syndrome 之**第一線治療 = 飲食調整(少量多餐、低單醣)**。

### 選項分析
- **(A) acarbose** — 非第一線。**acarbose 對『晚期(late)dumping/餐後低血糖』有效**,屬**飲食調整無效後**的藥物選項,非首選。
- **(B) metformin** — 錯誤。為降血糖藥,**不治療 dumping**;此症的低血糖是**餐後反應性低血糖**,加 metformin 無助甚至有害。
- **(C) metoclopramide** — 錯誤。為促胃排空藥;dumping 的問題是**胃排空過快**,促動力藥**反而加重**,方向相反。
- **(D) rifaximin** — 錯誤。用於**小腸細菌過度增生(SIBO)/肝性腦病/IBS-D**;本例切片與排空正常、症狀為傾食型,非 SIBO。
- **(E) 少量多餐** — **正確 → 本題答案**。**飲食調整為 dumping 的第一線**:**少量多餐、避免單醣/高糖、固液分開進食、增加纖維/蛋白**;多數病人靠飲食即可改善。

### 答案解析
題問「最恰當的治療」。減重手術後典型傾食症候群,**第一線為飲食調整(少量多餐、低單醣)**,故 **(E)** 為本題答案;acarbose/somatostatin analog 保留給飲食無效者。

### 核心知識點
- **早期 dumping(餐後 <1 小時)**:高滲透食糜快速入小腸 → 體液移位+腸道激素釋放 → **腹痛、腹瀉、心悸、流汗、潮紅、低血壓**。
- **晚期 dumping(餐後 1–3 小時)**:高糖快速吸收 → 過度胰島素 → **反應性低血糖**。
- **治療階梯**:**①飲食調整(第一線:少量多餐、低單醣、固液分食、富纖維/蛋白)→ ②acarbose(尤其晚期)→ ③somatostatin analog(octreotide;頑固者)→ ④手術修正**。
- **診斷**:症狀+**改良口服葡萄糖耐受試驗(OGTT,而非胃排空檢查)**;Sigstad score 輔助。

**題幹**  
60歲女性因解血便至急診，經輸液治療穩定生命徵象後安排大腸鏡及胃鏡檢查，結果並無發現可疑的出血病灶，下列敘述何者錯誤？

- (C) 可以是血管性病灶。
- (D) 患者若持續嚴重性出血可考慮Tc-RBC scan或angiography。
- (E) 若出血停止則考慮進行video capsule或enteroscopy檢查。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**消化道出血的分類術語與小腸(obscure)出血的處置流程**:60 歲女性**解血便(顯性出血,overt bleeding)**,輸液穩定後**胃鏡+大腸鏡皆未見病灶**。考點為正確區分 **occult vs overt、obscure GI bleeding** 與後續檢查路徑。需挑出**錯誤**敘述,陷阱在把「解血便(顯性)」誤稱為 occult(隱性)出血。

### 選項分析
- **(A) 此種出血型態稱為 occult GI bleeding** — **錯誤 → 本題答案**。病人有**肉眼可見血便(解血便)= overt(顯性)出血**;**occult(隱性)出血指肉眼看不見、僅糞便潛血陽性或缺鐵性貧血**。此處上下消化道內視鏡皆陰性而仍出血,正確術語為 **overt obscure GI bleeding(顯性不明原因出血)**,非 occult。把顯性叫成隱性是經典定義陷阱。
- **(B) 小腸是最可能的病灶位置** — 正確。上下內視鏡陰性的不明原因出血,**最可能來源是小腸**(血管異常、腫瘤、潰瘍、Meckel 憩室等)。
- **(C) 可以是血管性病灶** — 正確。**小腸血管擴張(angioectasia/AVM)** 是中老年小腸出血常見原因。
- **(D) 持續嚴重出血可考慮 Tc-RBC scan 或 angiography** — 正確。**活動性、持續出血**時,**核醫紅血球掃描(偵測下限低)或血管攝影(可同時栓塞)**有助定位/止血。
- **(E) 出血停止則考慮 video capsule 或 enteroscopy** — 正確。**穩定/出血停止**者,**膠囊內視鏡或小腸鏡(enteroscopy)** 為評估小腸的首選工具。

### 答案解析
題問「何者**錯誤**」。病人為**顯性(overt)血便且內視鏡陰性 → overt obscure GI bleeding**,而非 occult(隱性)出血,故 **(A)** 錯誤、即本題答案。

### 核心知識點
- **術語**:**occult(隱性)**=肉眼不可見(FOBT(+)/缺鐵貧血);**overt(顯性)**=肉眼可見(嘔血/黑便/血便);**obscure(不明原因)**=上下內視鏡找不到來源(可再分 overt/occult)。
- **小腸出血常見原因**:**血管擴張(angioectasia)、NSAID 潰瘍、腫瘤(GIST/類癌)、Meckel 憩室(年輕)、Dieulafoy**。
- **檢查流程**:穩定 → **膠囊內視鏡 → 小腸鏡(可治療)/CT enterography**;**持續活動出血/不穩 → CTA 或 Tc-RBC scan → 血管攝影栓塞**。
- **新名詞**:現多以 **small bowel bleeding** 取代 obscure(因膠囊/小腸鏡已能直視小腸)。
