**題幹**  
有關僵直性脊椎炎的血液及血清學檢查數據如下，請問何者最為正確?(1) ANA(+) (2) ESR上升( 3) CRP上升 (4) RF (+) (5)Anti-CCP (-) (6) HLA-B27(+) (7) ANCA(+)

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**僵直性脊椎炎(ankylosing spondylitis, AS)的血清學特徵**。AS 屬**血清陰性脊椎關節病變(seronegative spondyloarthropathy)**,核心觀念有三:(1)**HLA-B27 強烈相關**(台灣 AS 病人 90% 以上陽性);(2)為**自體發炎而非典型自體免疫病**,故 **ANA、RF、anti-CCP 多為陰性**;(3)活動期會有**全身發炎指標(ESR、CRP)上升**,但約 1/3 病人 ESR/CRP 可正常。把六、七個敘述逐一比對「正確組合」即可。

### 選項分析
- **(A) 1+2+3(ANA(+)、ESR↑、CRP↑)** — 錯誤。AS 屬血清陰性,**ANA 通常為陰性**,(1)錯,故整組不成立。ANA 陽性是 SLE/混合結締組織病的陷阱。
- **(B) 4+5+6(RF(+)、anti-CCP(-)、HLA-B27(+))** — 錯誤。AS **RF 應為陰性**(seronegative),(4)錯;RF(+)是把 RA 的特徵誤植。
- **(C) 2+4+7(ESR↑、RF(+)、ANCA(+))** — 錯誤。RF(+)、ANCA(+)皆非 AS 特徵;ANCA 對應血管炎,屬混淆項。
- **(D) 3+4+5(CRP↑、RF(+)、anti-CCP(-))** — 錯誤。CRP↑與 anti-CCP(-)雖正確,但夾帶 RF(+)即不成立。
- **(E) 3+5+6(CRP↑、anti-CCP(-)、HLA-B27(+))** — **正確 → 本題答案**。三項皆符合 AS:活動期 **CRP 上升**、**anti-CCP 陰性**(非 RA)、**HLA-B27 陽性**(最具代表性的遺傳標記),為唯一全對的組合。

### 答案解析
題問「何者最為正確」。AS 為血清陰性、HLA-B27 相關、活動期急性期反應物上升;唯一三項全符合者為 **CRP↑ + anti-CCP(-) + HLA-B27(+)**,故選 **(E)**。

### 核心知識點
- **AS 血清學**:**HLA-B27(+)約 90%**、**RF/anti-CCP/ANA 皆陰性**、活動期 **ESR/CRP 可↑(但 1/3 正常)**。
- 診斷不靠單一抽血;影像(薦腸關節炎,mNY 標準需 X 光雙側≥2 級或單側 3–4 級;早期用 **MRI 偵測骨髓水腫**)＋臨床(發炎性下背痛、晨僵>30 分、活動後改善)為主。
- **發炎性下背痛特徵**:年齡<40 歲發病、隱襲性、休息更痛、夜間痛、運動改善。
- CRP 升高者對 **TNF 抑制劑/IL-17 抑制劑**反應較佳,可作治療反應預測。

### 105-140 ・ 正解 C ・ status: draft

**題幹**  
僵直性脊椎炎(Ankylosing spondylitis ) (AS)之發生與基因有密切關係，下列基因與AS之發生有關? 1. HLA-B27 2. HLA-DR4 3. HLA-B8 4. IL-23R 5. HLA-B47

- (A) 1+2
- (B) 1+3
- (C) 1+4 ← 正解
- (D) 1+5
- (E) 3+5

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**僵直性脊椎炎(AS)的遺傳易感基因**。AS 是高度遺傳性疾病(遺傳貢獻度逾 90%)。最重要的是 **HLA-B27(MHC class I)**,佔遺傳風險約 20–25%;其餘由多個非 MHC 基因構成,其中以 **IL-23/IL-17 軸**最具代表——**IL23R** 是繼 HLA-B27 之後最早被 GWAS(WTCCC/TASC)確認的 AS 易感基因之一,並衍生出 IL-17/IL-23 抑制劑等治療。需從清單挑出真正與 AS 相關者。

### 選項分析
- **(1) HLA-B27** — 正確。**AS 最主要的遺傳標記**,台灣病人陽性率>90%。
- **(2) HLA-DR4** — 錯誤(陷阱)。HLA-DR4(DRB1 共有抗原決定位)是**類風濕性關節炎(RA)**的相關基因,非 AS。
- **(3) HLA-B8** — 錯誤。HLA-B8(B8-DR3 單倍型)與**自體免疫病如 SLE、重症肌無力、第一型糖尿病、自體免疫肝炎**相關,非 AS。
- **(4) IL-23R** — 正確。**IL23R 多型性經 GWAS 證實與 AS 相關**,呼應 IL-23/IL-17 致病軸與生物製劑標靶。
- **(5) HLA-B47** — 錯誤。與 AS 無關;HLA-B 系列中具意義的是 B27(及亞型如 B*2705/2704)。

### 答案解析
題問與 AS 發生「有關」的基因。正確者為 **(1) HLA-B27 與 (4) IL-23R**,組合即 **(C) 1+4**,為本題答案。DR4 屬 RA、B8 屬其他自體免疫病,皆為干擾。

### 核心知識點
- **AS 遺傳**:**HLA-B27** 為主(風險約佔 20–25%);非 MHC 基因以 **IL23R、ERAP1、IL12B、IL1R2** 等為代表(IL-23/IL-17 軸)。
- **ERAP1 與 HLA-B27 有交互作用**(僅在 B27(+)者影響風險),為機轉重點。
- 對照記憶:**HLA-B27→AS/反應性關節炎/葡萄膜炎**;**HLA-DR4→RA**;**HLA-B8-DR3→SLE 等**;**HLA-B5(B51)→Behçet**;**HLA-B57→abacavir 過敏、HIV 控制**。
- IL-23/IL-17 軸的臨床對應:**secukinumab/ixekizumab(抗 IL-17A)**、TNF 抑制劑為 AS 生物製劑主力。

**題幹**  
類風溼性關節炎(Rheumatoid arthritis)為嚴重之關節炎，下列敘述為正確? 1. 抗腫瘤壞死生物製劑為常用之治療 2. 常侵犯遠端指關節 3. 抗環瓜氨酸抗體(anti-CCP)為其特徵之一 4. 常發生在HLA-B8之病 5. 抽菸無關

- (A) 1+2
- (B) 1+3 ← 正解
- (C) 1+4
- (D) 1+5
- (E) 3+5

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**類風濕性關節炎(rheumatoid arthritis, RA)的臨床與免疫特徵**。RA 為對稱性、侵犯**小關節**的慢性發炎性多關節炎,核心觀念:(1)**典型侵犯近端指間關節(PIP)、掌指關節(MCP)、腕關節,而非遠端指間關節(DIP)**(DIP 是 OA/乾癬性關節炎的領域);(2)**anti-CCP(抗環瓜氨酸抗體)是高特異性標記**且預後較差;(3)**抽菸**經由瓜氨酸化是明確環境危險因子;(4)遺傳與 **HLA-DR4(共有抗原決定位)**相關,非 B8;(5)中重度活動且預後不良者可用 **TNF 抑制劑**等生物製劑。

### 選項分析
- **(1) 抗腫瘤壞死(TNF)生物製劑為常用治療** — 正確。csDMARD(MTX)反應不足者,**TNF 抑制劑為常用生物製劑**之一。
- **(2) 常侵犯遠端指關節(DIP)** — 錯誤(經典陷阱)。RA **典型侵犯 MCP/PIP/腕**,**DIP 多保留**;DIP 受侵應想到 **OA(Heberden 結節)或乾癬性關節炎**。
- **(3) anti-CCP 為其特徵之一** — 正確。**anti-CCP 特異性>95%**,陽性者侵蝕性病程與預後較差,屬 RA 標記。
- **(4) 常發生在 HLA-B8 之病人** — 錯誤。RA 與 **HLA-DR4/DRB1 共有抗原決定位**相關;**HLA-B8 屬 SLE 等自體免疫病**,為混淆。
- **(5) 與抽菸無關** — 錯誤。**抽菸是 RA(尤其 anti-CCP(+))明確危險因子**,並降低治療反應。

### 答案解析
題問「何者正確」。正確敘述為 **(1) 生物製劑常用** 與 **(3) anti-CCP 為特徵**,組合即 **(B) 1+3**,為本題答案。DIP 侵犯、HLA-B8、與抽菸無關皆錯。

### 核心知識點
- **RA 關節分布**:對稱性、**MCP/PIP/腕/MTP**;**DIP 保留**(對比 OA、PsA 侵 DIP)。
- **血清學**:**RF(敏感)＋anti-CCP(特異>95%)**;雙陽性預後較差。
- **遺傳/環境**:**HLA-DRB1 共有抗原決定位(shared epitope)**＋**抽菸**→經瓜氨酸化致病。
- **治療(treat-to-target)**:確診即起 **MTX(口服 7.5→up to 25 mg/週,加葉酸)**±短期低劑量類固醇橋接;3–6 個月未達標依預後因子加 **bDMARD(TNFi、IL-6Ri、abatacept、rituximab)或 JAKi**。
- **2010 ACR/EULAR 分類**最重視**受侵關節數目與大小(小關節權重高)**＋血清學＋急性期反應物＋病程≥6 週。

下列為僵直性脊椎炎(AS)常見之關節外之症狀或疾病 1. 前葡萄膜炎(彩虹炎) 2. Giant cell arteritis ( temporal arteritis) 3. 過敏性皮膚炎 4. 腸炎 ( inflammatory bowel disease) 5. 肺癌

- (A) 1+2
- (B) 1+3
- (C) 1+4 ← 正解
- (D) 1+5
- (E) 2+4

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**僵直性脊椎炎(AS)的關節外表現(extra-articular manifestations)**。AS 屬脊椎關節病變,關節外表現以「**A 字訣**」記憶最常考:**Anterior uveitis(前葡萄膜炎,即彩虹炎)**、**Aortitis/Aortic regurgitation(主動脈炎/閉鎖不全)**、**Apical lung fibrosis(肺尖纖維化)**、**Amyloidosis**、**IBD(發炎性腸道疾病)**。其中**急性前葡萄膜炎是最常見**(約 25–40%),且 HLA-B27(+)者風險更高;與 IBD 同屬 SpA 家族重疊。需挑出真正屬 AS 的兩項。

### 選項分析
- **(1) 前葡萄膜炎(彩虹炎)** — 正確。**AS 最常見的關節外表現**,常為單側、復發性、急性紅眼痛、畏光,HLA-B27 相關。
- **(2) Giant cell arteritis(巨細胞/顳動脈炎)** — 錯誤(陷阱)。GCA 是>50 歲的大血管炎,與 **PMR** 相關,與 AS 無關。
- **(3) 過敏性皮膚炎** — 錯誤。AS 的皮膚關聯是**乾癬**(脊椎關節病變家族),非過敏性皮膚炎。
- **(4) 腸炎(inflammatory bowel disease)** — 正確。**IBD 與 AS 同屬脊椎關節病變家族**,可重疊(腸病性關節炎),次臨床腸道發炎更常見。
- **(5) 肺癌** — 錯誤。AS 的肺部表現是**肺尖纖維化、胸廓擴張受限**,非肺癌。

### 答案解析
題問 AS 常見的關節外症狀/疾病。正確者為 **(1) 前葡萄膜炎** 與 **(4) IBD**,組合即 **(C) 1+4**,為本題答案。GCA、過敏性皮膚炎、肺癌皆非 AS。

### 核心知識點
- **AS 關節外(SpA 家族重疊)**:**前葡萄膜炎(最常見)、乾癬、IBD**;另有**主動脈瓣閉鎖不全/傳導阻滯、肺尖纖維化、續發性類澱粉沉積、骨質疏鬆致脊椎骨折**。
- **前葡萄膜炎**:急性、單側、復發、HLA-B27(+),需眼科處置;TNF 單株抗體(adalimumab/infliximab)可降復發,**etanercept 預防葡萄膜炎效果較差**。
- 對比:**GCA→PMR/顳動脈、年長**;**過敏性皮膚炎→異位性體質**,皆非 SpA。
- SpA 家族:AS、乾癬性關節炎、反應性關節炎、腸病性關節炎、未分化型——共同特徵為附著點炎、指趾炎、葡萄膜炎、HLA-B27 相關。

根據2010美國風濕病醫學會與歐洲風濕病聯盟制定類風溼性關節炎之診斷分類準則，下列何者為較重要之診斷要件

- (B) 侵犯小關節 ← 正解
- (C) 反應蛋白濃度(C-reactive protein)
- (D) 紅血球下降速率(ESR)
- (E) 發病時間之長短

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**2010 ACR/EULAR 類風濕性關節炎分類準則的權重結構**。此準則適用於「至少有一個關節滑膜炎且無法以其他疾病解釋」者,總分≥6/10 可分類為 RA。四大領域與配分:**關節受侵(0–5 分,權重最高)、血清學(RF/anti-CCP,0–3)、急性期反應物(ESR/CRP,0–1)、症狀持續時間(0–1)**。其中**「受侵關節的數目與大小」配分最高**,且**小關節(尤其多顆小關節)權重高於大關節**——這正是本題要點。

### 選項分析
- **(A) 侵犯大關節** — 錯誤。關節領域中**僅 1 顆大關節得 0 分、2–10 顆大關節得 1 分**,權重低。
- **(B) 侵犯小關節** — **正確 → 本題答案**。**小關節受侵權重最高**:1–3 顆小關節 2 分、**4–10 顆小關節 3 分、>10 顆(含至少 1 小關節)5 分**,是達到分類門檻最關鍵的要件。
- **(C) CRP 濃度** — 錯誤。急性期反應物(CRP 或 ESR)**異常僅得 1 分**,權重低。
- **(D) ESR(紅血球沉降速率)** — 錯誤。同 (C),ESR 與 CRP 合計**最多 1 分**。
- **(E) 發病時間長短** — 錯誤。症狀持續**≥6 週僅得 1 分**,非關鍵。

### 答案解析
題問「較重要之診斷要件」。2010 準則中**關節領域(尤其小關節數目)配分最高(可達 5 分)**,故 **(B) 侵犯小關節**為本題答案;血清學次之(最多 3),急性期反應物與病程各僅 1 分。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題情境:**硬皮症(systemic sclerosis, SSc)多年、新近運動時喘**。硬皮症病人喘的兩大致命併發症為**間質性肺病(ILD)**與**肺動脈高壓(PAH)**,二者皆為 SSc 主要死因。題目強調「**走路時會喘**」(運動性呼吸困難)且病史久,**limited cutaneous SSc(CREST,常 anti-centromere(+))特別好發 PAH**。臨床路徑為**先以非侵入篩檢(心臟超音波估計肺動脈收縮壓)**,異常者再以**右心導管(金標準)確診**——故「測肺動脈血壓」是最適當的下一步。

### 選項分析
- **(A) 檢查 anti-Scl-70 抗體** — 錯誤。anti-Scl-70(anti-topoisomerase I)用於分類與**評估 ILD/瀰漫型風險**,但對「喘」的當下診斷與處置幫助有限,且抗體已可能早知。

- **(C) 抗磷脂質抗體** — 錯誤。與 SSc 喘的主因(PAH/ILD)無直接關係。
- **(D) 測心肌酵素(CK-MB)** — 錯誤。CK-MB 用於急性心肌梗塞;SSc 喘首先想肺血管/肺實質,非缺血。
- **(E) 肺部切片** — 錯誤。評估 SSc-ILD 用**高解析度胸部 CT(HRCT)＋肺功能(DLCO/FVC)**,非首選切片(侵入且非必要)。

### 核心知識點
- **SSc 喘鑑別**:**PAH**(limited/CREST、anti-centromere、DLCO 不成比例下降)vs **ILD**(diffuse、anti-Scl-70、HRCT 毛玻璃/網狀、FVC↓)。

- **SSc 抗體對應**:**anti-centromere→limited/PAH**;**anti-Scl-70→diffuse/ILD**;**anti-RNA pol III→瀰漫型/腎危象(scleroderma renal crisis,用 ACEi)**。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題為典型**異位性皮膚炎(atopic dermatitis, AD)**:**幼年發病(3 歲起)的慢性搔癢性濕疹、過敏家族史(atopy)、好發部位呈苔蘚化(臉部、肘窩屈側)、血清總 IgE 顯著升高(3750 IU/mL)**,且病程慢性反覆、外用類固醇有效但無法根治。這些特徵組合(慢性、屈側、苔蘚化、高 IgE、家族史)幾乎是 AD 的考試標準畫像。

### 選項分析
- **(A) Psoriasis(乾癬)** — 錯誤。乾癬為**伸側(肘、膝)邊界清楚的銀白色脫屑斑塊**、Auspitz sign、與 IgE 無關;此為「伸側 vs 屈側」與「脫屑斑塊 vs 苔蘚化濕疹」的鑑別陷阱。
- **(B) Chronic urticaria(慢性蕁麻疹)** — 錯誤。蕁麻疹為**短暫(<24 小時)、會移位的膨疹/風團**,不留苔蘚化痕跡,且非自幼持續的固定濕疹斑。
- **(C) Atopic dermatitis(異位性皮膚炎)** — **正確 → 本題答案**。符合幼年起病、慢性屈側苔蘚化、過敏家族史、極高總 IgE 之 atopic march 特徵;機轉為**皮膚屏障缺損(filaggrin 突變)＋Th2 偏移(IL-4/IL-13)**。
- **(D) Lichen planus(扁平苔蘚)** — 錯誤。為**紫紅色、多角形、扁平丘疹(6P)＋Wickham striae**,好發手腕屈側與口腔,屬 T 細胞媒介、與高 IgE 無關。
- **(E) Contact dermatitis(接觸性皮膚炎)** — 錯誤。為**接觸特定過敏原/刺激物的局部濕疹**,分布對應接觸部位、移除過敏原即改善,不會自 3 歲起全身慢性反覆且伴極高 IgE。

### 答案解析
題問「最可能診斷」。幼年慢性、屈側苔蘚化、atopy 家族史、總 IgE 3750 IU/mL → 最符合 **(C) Atopic dermatitis**。

### 核心知識點
- **AD 三大線索**:**慢性復發性搔癢濕疹、典型分布(嬰兒臉/伸側→兒童及成人屈側)、個人/家族 atopy 史**;多數總 IgE 升高(但非診斷必需)。
- **機轉**:**filaggrin 屏障缺損＋Th2(IL-4/IL-13/IL-31 致癢)**;屬 **atopic march**(AD→食物過敏→過敏性鼻炎→氣喘)。
- **階梯治療**:基礎**保濕＋避免誘因**;發作期**外用類固醇 / TCI(tacrolimus、pimecrolimus)**;中重度/難治用 **dupilumab(抗 IL-4Rα)** 或 **JAK 抑制劑(upadacitinib、abrocitinib)**。

- **(A) Polyarteritis nodosa(PAN)** — 錯誤。PAN 為**中血管壞死性血管炎**,典型**不侵犯肺、ANCA 多陰性**,以腎血管動脈瘤/梗塞、腸繫膜、皮膚、周邊神經(mononeuritis multiplex)為主,與本例肺結節＋cANCA 不符。
- **(B) Eosinophilic granulomatosis with polyangiitis(EGPA)** — 錯誤。EGPA 以**氣喘、過敏性鼻炎、顯著嗜酸性球增多**為特徵,ANCA 多為 **pANCA(MPO)**且陽性率較低;本例血液常規正常、無嗜酸性球增多、為 cANCA,不符。
- **(C) Sarcoidosis(類肉瘤病)** — 錯誤。以**雙側肺門淋巴結腫大、非乾酪性肉芽腫**為主,**不會 cANCA 陽性、不以鼻出血/壞死性腎炎表現**。

- **(E) Paragonimiasis(肺吸蟲病)** — 錯誤。寄生蟲感染可致喀血與肺病灶,但**伴嗜酸性球增多、有生食蝲蛄/淡水蟹史**,且不會 cANCA 陽性與壞死性腎炎,屬干擾項。

### 答案解析
題問「最可能診斷」。**ENT＋肺＋腎三器官＋cANCA(PR3)陽性** → 最符合 **(D) GPA**。

### 核心知識點
- **GPA 三聯**:**上呼吸道(鼻竇炎、鞍鼻、鼻出血)＋肺(結節/空洞、喀血、肺泡出血)＋腎(pauci-immune 新月體腎炎)**;**cANCA/anti-PR3** 為其標記(MPA 多為 pANCA/MPO)。
- **確診**:臨床＋ANCA＋**組織切片(壞死性肉芽腫性血管炎)**;2022 ACR/EULAR 分類以 PR3-ANCA、鼻/竇病變、肺結節等加權計分。
- **治療(分期)**:**嚴重/器官威脅之誘導**用 **rituximab 或 cyclophosphamide＋高劑量 glucocorticoid**(嚴重者考慮血漿置換);近年 **avacopan(C5a 受體拮抗)** 可助減類固醇。**維持**用 rituximab 或 azathioprine。
