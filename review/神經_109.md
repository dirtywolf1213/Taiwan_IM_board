### 本題觀念
同一工廠多人出現 parkinsonism，要想到職業暴露造成的 manganism。Manganese exposure 可見於 welding、mining、metal processing 等，造成 bradykinesia、步態障礙、平衡差與 Parkinson-like syndrome。

### 選項分析

### 核心知識點
- Manganese toxicity：manganism、parkinsonism、職業暴露。
- Welding fumes 是常見暴露場景。
- 有群聚神經症狀時，要問職業史與環境暴露。

**題幹**  
75歲女性，過去有高血壓、糖尿病，突發講話詞不達意，送至急診後腦部磁振照影顯示左側頂葉部急性梗塞，則會出現下列症狀何者除外？

- (A) 左右混淆（right left disorientation）
- (B) 手指辨識困難（finger agnosia）
- (C) 不會穿衣（dressing apraxia） ← 正解
- (D) 計算困難（acalculia）
- (E) 書寫困難（agraphia）

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
左側頂葉，尤其 dominant inferior parietal lobule/angular gyrus 病灶，可造成 Gerstmann syndrome：左右混淆、finger agnosia、acalculia、agraphia。Dressing apraxia 較常與右側 parietal lobe 或非優勢半球空間處理障礙相關。

### 選項分析
- **(A) 左右混淆**：可見於 Gerstmann syndrome。
- **(B) 手指辨識困難**：可見於 Gerstmann syndrome。
- **(C) 不會穿衣**：**除外 -> 本題答案**。dressing apraxia 較常為右頂葉/非優勢半球病灶。
- **(D) 計算困難**：可見於 Gerstmann syndrome。
- **(E) 書寫困難**：可見於 Gerstmann syndrome。

### 答案解析
題幹指定左側頂葉急性梗塞，最典型聯想到 dominant parietal Gerstmann syndrome。穿衣失用較偏 visuospatial/body schema impairment，常見於右頂葉病灶，因此為除外。

### 核心知識點
- Gerstmann syndrome：agraphia、acalculia、finger agnosia、left-right disorientation。
- Dominant parietal lobe：語言相關、計算、書寫、左右辨識。
- Non-dominant parietal lobe：neglect、dressing apraxia、constructional apraxia。

### 109-181 ・ 正解 D ・ status: draft

**題幹**  
65歲男性，2年前開始有焦慮、情緒低落、睡眠差症狀，漸漸發生步態不穩僵硬、記憶衰退，偶有視幻覺與妄想症狀，最可能是下列何診斷？

- (A) 皮質基底核退化（corticobasal ganglion degeneration）
- (B) 多系統萎縮（multiple system atrophy）
- (C) 漸進性上核麻痺（progressive supranuclear palsy）
- (D) 路易體失智症（dementia with Lewy body） ← 正解
- (E) 額顳葉失智症（frontotemporal dementia）

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考 dementia with Lewy bodies (DLB) 的臨床辨識。中老年人出現失智合併早期 visual hallucination、妄想、parkinsonism、睡眠/精神症狀時，要想到 DLB；若 parkinsonism 已存在多年後才失智，才較偏 Parkinson disease dementia 的「1-year rule」。

### 選項分析
- **(A) 皮質基底核退化（corticobasal ganglion degeneration）**：錯誤。Corticobasal syndrome 常是不對稱僵硬、失用、皮質感覺缺損或 alien limb，題幹的視幻覺與失智-parkinsonism 組合不典型。
- **(B) 多系統萎縮（multiple system atrophy）**：錯誤。MSA 以 autonomic failure、小腦症狀或 parkinsonism 為主，早期明顯失智與視幻覺不是核心特徵。
- **(C) 漸進性上核麻痺（progressive supranuclear palsy）**：錯誤。PSP 典型是早期跌倒、軸性僵硬、垂直凝視障礙，題幹未提供這些線索。
- **(D) 路易體失智症（dementia with Lewy body）**：**正確 -> 本題答案**。認知退化合併步態僵硬/parkinsonism、視幻覺與妄想，最符合 DLB。
- **(E) 額顳葉失智症（frontotemporal dementia）**：錯誤。FTD 以人格行為改變、去抑制、冷漠或語言退化為主，視幻覺較不典型。

### 答案解析
題幹的組合是「失智 + parkinsonism + recurrent visual hallucination/psychosis」，這是 DLB 的典型考點。皮質基底核退化、MSA、PSP 都可造成 parkinsonism，但各自有更突出的神經定位特色；FTD 則更偏行為或語言型退化。因此答案為 **(D)**。

### 核心知識點
- DLB 核心特徵：fluctuating cognition、recurrent visual hallucinations、REM sleep behavior disorder、spontaneous parkinsonism。
- DLB 病人對 antipsychotics 可能特別敏感，若需治療精神症狀要非常謹慎。
- 考場陷阱：看到「僵硬」不要直接選 MSA/PSP，要看是否合併早期失智與視幻覺。

### 參考資料
- McKeith IG, et al. Diagnosis and management of dementia with Lewy bodies: fourth consensus report. *Neurology*. 2017. https://pubmed.ncbi.nlm.nih.gov/28592453/
- National Institute on Aging. Lewy Body Dementia. https://www.nia.nih.gov/health/lewy-body-dementia

</details>

### 109-182 ・ 正解 D ・ status: draft

**題幹**  
52歲女性，原即有控制不很好的高血壓，某日與他人激烈爭吵後，突發劇烈頭痛，合併意識有些混沌，緊急送醫後磁振照影顯示腦部的兩側枕葉有腫脹，但無梗塞或出血。關於此疾病，下列處置何者較正確？

- (A) 應優先考慮腦血管炎（cerebral vasculitis）
- (B) 應考慮導管是血管攝影（catheter angiography）
- (C) 應給予類固醇治療
- (D) 應給予靜脈鈣離子阻斷劑 ← 正解
- (E) 應給予抗癲癇藥物

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
高血壓後急性頭痛、意識混沌，MRI 兩側枕葉腫脹但無梗塞或出血，最符合 posterior reversible encephalopathy syndrome (PRES)。PRES 的影像重點是 posterior-predominant vasogenic edema，治療核心是移除誘因與控制血壓。

### 選項分析
- **(A) 應優先考慮腦血管炎（cerebral vasculitis）**：錯誤。血管炎可列入鑑別，但題幹「高血壓 + bilateral occipital edema」更典型是 PRES。
- **(B) 應考慮導管是血管攝影（catheter angiography）**：錯誤。PRES 通常不以侵入性血管攝影作為優先處置。
- **(C) 應給予類固醇治療**：錯誤。類固醇不是 PRES 標準治療，除非另有免疫疾病、血管炎等適應症。
- **(D) 應給予靜脈鈣離子阻斷劑**：**正確 -> 本題答案**。急性血壓控制可使用可滴定 IV antihypertensive，例如 nicardipine/clevidipine 這類 calcium channel blocker。
- **(E) 應給予抗癲癇藥物**：錯誤。若發生 seizure 當然要治療，但題幹未描述癲癇；本題優先是控壓。

### 答案解析
PRES 常與 uncontrolled hypertension、子癇、免疫抑制藥或腎病相關，臨床可有頭痛、視覺症狀、意識改變與癲癇。此題沒有梗塞出血，且病灶位於雙側枕葉，最應先處理血壓；答案鍵的 IV CCB 可理解為急性可滴定降壓治療，因此選 **(D)**。

### 核心知識點
- PRES 影像常見 posterior vasogenic edema，尤其 parieto-occipital region。
- 治療：控制血壓、移除誘因、處理癲癇；不是常規類固醇或血管攝影。
- 易混淆：RCVS 也會雷擊頭痛並可用 CCB，但題幹強調雙側枕葉水腫，考點更像 PRES。

### 參考資料
- StatPearls. Posterior Reversible Encephalopathy Syndrome. https://www.ncbi.nlm.nih.gov/books/NBK554492/
- Triplett JD, et al. Posterior reversible encephalopathy syndrome. *Practical Neurology*. 2022. https://pn.bmj.com/content/22/3/183

</details>

**題幹**  
50歲男性，有酒癮與曾有酒精戒斷症狀，因急性神經症狀，包括複視（diplopia）、步履不穩（ataxia）、記憶喪失（memory loss）、淡漠（apathy）、混沌（confusion）等入院治療，初步診斷為維生素B1缺乏，給予thiamine治療，下列那一個症狀恢復較快？

- (B) 步履不穩
- (C) 記憶喪失
- (D) 淡漠
- (E) 混沌 0.

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
Wernicke encephalopathy 是 thiamine deficiency 造成的急症，典型三聯徵為 ophthalmoplegia/nystagmus、ataxia、confusion，但三者不一定全出現。治療後眼球運動異常通常改善最快，記憶障礙可能最慢且可能殘留 Korsakoff syndrome。

### 選項分析
- **(A) 複視**：**正確 -> 本題答案**。複視來自眼球運動異常，補充 thiamine 後常在數小時到數日內改善。
- **(B) 步履不穩**：錯誤。Ataxia 可改善，但常比眼徵慢，也可能殘留。
- **(C) 記憶喪失**：錯誤。記憶障礙是最容易殘留的後遺症之一。
- **(D) 淡漠**：錯誤。淡漠屬精神狀態變化，可改善但不是最快恢復的典型症狀。
- **(E) 混沌 0.**：錯誤。Confusion 可改善，但題目問較快恢復，仍以眼球運動異常最典型；選項尾端數字應為 OCR 雜訊。

### 答案解析
酒癮病人出現 diplopia、ataxia、confusion、memory loss，須先給 thiamine，且臨床上應在給葡萄糖前或同時補充。眼球運動異常最常快速改善，因此答案為 **(A)**。

### 核心知識點
- Wernicke 不可等 MRI 或完整三聯徵才治療，疑似即給 parenteral thiamine。
- 眼徵最快改善；步態與記憶恢復較慢，Korsakoff memory impairment 可能不可逆。
- 考場陷阱：酒癮 + 複視/共濟失調/混亂，不要只當作戒斷或中毒。

### 本題觀念
Cluster headache 治療分為急性止痛與預防。急性發作首選高流量 100% oxygen 或皮下/鼻噴 triptan；verapamil 是週期性發作的預防一線。Indomethacin 則更典型用於 paroxysmal hemicrania，不是 cluster headache 主治療。

### 選項分析
- **(A) (1)+(3)**：**正確 -> 本題答案**。氧氣可急性緩解；verapamil 是 cluster bout 預防主力。題幹雖寫急性發作，但選項設計顯然把急性與預防合併考。
- **(B) (2)+(4)**：錯誤。口服 sumatriptan 起效較慢，不是 cluster 急性首選；indomethacin 不是 cluster headache 標準治療。
- **(C) (1)+(4)**：錯誤。氧氣正確，但 indomethacin 不符合 cluster headache。
- **(D) (2)+(3)**：錯誤。Verapamil 對預防正確，但口服 sumatriptan 不如皮下或鼻噴劑型適合急性 cluster attack。
- **(E) (1)+(2)+(3)+(4)**：錯誤。包含口服 sumatriptan 與 indomethacin 兩個不佳選項。

### 答案解析
本題答案鍵為 **(A)**，可理解為「急性發作時用氧氣，反覆冬季 cluster bout 則用 verapamil 預防」。嚴格來說，verapamil 不是 acute abortive therapy；這正是本題的文字陷阱，讀者應把題幹的「每年冬季連續 5-6 週」納入判讀。

### 核心知識點
- Acute cluster attack：100% oxygen 12-15 L/min 以上、subcutaneous/intranasal sumatriptan 或 zolmitriptan。
- Prevention：verapamil 一線，需注意 ECG 監測與劑量調整。
- Indomethacin-responsive headache 典型是 paroxysmal hemicrania/hemicrania continua。
