### 113-074 ・ 正解 D ・ status: draft

**題幹**  
病人因倦怠及食慾不振而來就醫，抽血檢查結果如下：Free T4=1.00ng/dL (0.7-1.48)，TSH=1.85mU/L (0.35-4.94)，BUN=15mg/dL (<25)，Cr=0.6mg/dL (<1.2)，ALT=450 U/L (<41)，AST=350 U/L (<31)，ALP=650 U/L (<104)，γ-GT=790 U/L (<50)，Bil(T)/(D)=2.5/1.85 mg/dL (<1.1/<0.4)，Albumin=3.9 g/dL (>3.5)，Prothrombin time INR=1.05，Anti-HCV (+)，HCV RNA: Not detected，HBsAg (+)，HBV DNA=500 IU/mL。下列敘述何者正確？

- (A) 病人有hypothyroidism。
- (B) 病人有急性肝炎發作，應與B型肝炎病毒有關。
- (C) 病人有急性肝炎發作，應與C型肝炎病毒有關。
- (D) 病人有急性肝炎發作，其病因很可能為膽汁滯留性肝病 (cholestatic liver disease)。 ← 正解
- (E) 病人已有肝臟失代償現象 (liver decompensation)。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題以「倦怠、食慾不振」搭配一組肝膽生化判讀,考兩件事:(1)甲狀腺功能是否異常;(2)肝損傷型態(hepatocellular vs cholestatic)與病毒性肝炎血清學的整合判讀。判讀肝功能要先分型:**ALT/AST 為主、ALP/γ-GT 相對輕度↑ → hepatocellular**;**ALP/γ-GT 明顯↑、膽紅素↑而 ALT/AST 相對較低 → cholestatic**。本例 ALT 450、AST 350,而 **ALP 650、γ-GT 790** 同步大幅升高,屬**混合型但膽汁滯留色彩濃厚**;Albumin 3.9、INR 1.05 皆正常 → **肝臟合成功能保留,無失代償**。

### 選項分析
- **(A) 病人有 hypothyroidism** — 錯誤。Free T4 1.00(正常)、TSH 1.85(正常),甲狀腺功能完全正常;倦怠、食慾不振是非特異症狀,不能據此診斷甲狀腺低下。此為「症狀聯想陷阱」。
- **(B) 急性肝炎發作與 B 肝有關** — 錯誤。HBsAg(+)但 **HBV DNA 僅 500 IU/mL**(極低病毒量,遠低於 B 肝發作常見的 ≥2,000–20,000 IU/mL),不足以解釋 ALT 450 的明顯肝損;單純帶原者不能歸因為急性發作主因。
- **(C) 急性肝炎發作與 C 肝有關** — 錯誤。Anti-HCV(+)代表曾感染,但 **HCV RNA: Not detected**(病毒已清除/治癒),無複製即無活動性 C 肝炎,不可能是現行肝炎主因。此為「抗體陽性≠現症感染」經典陷阱。
- **(D) 急性肝炎發作,病因很可能為膽汁滯留性肝病** — **正確 → 本題答案**。在排除 B/C 肝病毒活動後,生化呈現 **ALP 650(>6×ULN)、γ-GT 790(>15×ULN)、直接型膽紅素為主(D-bil 1.85 占 T-bil 2.5 的多數)**,屬膽汁滯留型;γ-GT 與 ALP 同步↑亦確認來源為肝膽而非骨骼。
- **(E) 已有肝臟失代償現象** — 錯誤。失代償的指標是**合成功能下降(Albumin↓、INR/PT 延長)或併發症(腹水、肝性腦病、黃疸性凝血異常)**;本例 **Albumin 3.9 g/dL 正常、INR 1.05 正常**,僅膽紅素輕度↑,不構成 decompensation。

### 答案解析
題問「何者正確」。甲狀腺正常排除(A);B 肝病毒量極低、C 肝 RNA 測不到,皆非肝炎主因,排除(B)(C);Albumin/INR 正常排除失代償(E)。生化以 **ALP/γ-GT/直接膽紅素為主的膽汁滯留型**最符合,故選 **(D)**。

### 核心知識點
- **肝損型態判讀(R 值)**:R = (ALT/ULN)÷(ALP/ULN)。**R ≥ 5 hepatocellular、R ≤ 2 cholestatic、2–5 混合型**。本例 ALT 450/41≈11、ALP 650/104≈6.25,R≈1.75 → **cholestatic / 混合偏膽滯**。
- **γ-GT 的用途**:ALP↑時用 γ-GT 確認來源——**γ-GT 同步↑ → 肝膽源;γ-GT 正常而 ALP↑ → 骨骼/妊娠源**。
- **病毒血清學陷阱**:**Anti-HCV(+) + HCV RNA 未測到 = 既往感染/已清除或治癒**,非現症;**HBsAg(+)但 HBV DNA 極低**多為不活動帶原,B 肝發作通常伴隨**較高病毒量與 ALT flare**。
- **肝失代償判定**:看**合成功能(Albumin、PT/INR)與併發症(腹水、HE、靜脈曲張出血)**,而非單看膽紅素或轉胺酶高低。
- **甲狀腺**:Free T4 與 TSH 皆正常即可排除原發/中樞甲狀腺低下,不被非特異症狀誤導。

### 113-075 ・ 正解 D ・ status: draft

**題幹**  
一位74歲糖尿病女性患者合併慢性腎臟病第四期，長期服用口服降血糖用藥搭配胰島素控制，由於意識不清醒被送到急診就醫，經檢查有代謝性酸中毒(metabolic acidosis)，其陰離子隙(anion gap)為26 mmol/L，經積極使用碳酸氫鈉（sodium bicarbonate)矯正，仍無法控正，下列何種用口服降血糖藥物可能為誘發的原因？

- (A) 二肽基肽酶-4抑制劑（Dipeptidyl peptidase-4 inhibitor）。
- (B) 過氧化物酶體增殖物活化受體γ（PPAR-γ）。
- (C) α-葡萄糖苷酶抑制劑(α-glucosidase inhibitor)。
- (D) 二甲雙胍類降血糖藥物 (Metformin)。 ← 正解
- (E) 磺醯基尿素(sulfonylurea)。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考糖尿病用藥相關的**高陰離子隙代謝性酸中毒(high anion gap metabolic acidosis, AG 26)**鑑別。74 歲、CKD stage 4(eGFR 約 15–30)、口服降糖藥+胰島素、意識不清、AG 升高且補碳酸氫鈉難矯正——這是 **metformin-associated lactic acidosis(MALA)** 的經典情境:腎功能不全使 metformin 蓄積,抑制粒線體呼吸鏈與肝糖質新生,造成乳酸堆積。

### 選項分析
- **(A) DPP-4 抑制劑** — 不會。DPP-4i 經由增加 incretin 促胰島素分泌,**極少低血糖、不引起乳酸中毒**;部分(如 sitagliptin)CKD 需減量但安全性高。
- **(B) PPAR-γ(TZD,如 pioglitazone)** — 不會。機轉為增加胰島素敏感性;副作用是**水腫、心衰惡化、體重增加、骨折、膀胱癌風險**,與高 AG 酸中毒無關。
- **(C) α-glucosidase 抑制劑(acarbose)** — 不會。延緩腸道碳水吸收,副作用主要為**腹脹、脹氣、腹瀉**,不致乳酸中毒。
- **(D) Metformin** — **正確 → 本題答案**。**MALA** 雖罕見但死亡率高;**CKD stage 4(eGFR <30)為禁忌**,藥物經腎排除受阻而蓄積,抑制粒線體致乳酸生成↑、清除↓,造成**頑固性高 AG 代謝性酸中毒**,常需血液透析清除。
- **(E) Sulfonylurea** — 不會。SU 刺激 β 細胞分泌胰島素,主要風險是**低血糖**(尤其老人、腎功能差時延長),而非乳酸中毒;低血糖本身不造成高 AG 酸中毒。

### 答案解析
題問哪一種口服降糖藥可能誘發此頑固性高 AG 代謝性酸中毒。在 CKD stage 4 病人,**metformin 蓄積 → 乳酸中毒**最符合,故選 **(D)**。

### 核心知識點
- **Metformin 禁忌/停藥**:**eGFR <30 禁用;30–45 不新增、已用者減量並監測**;**急性病、敗血症、缺氧、顯影劑檢查、手術前後暫停**(降低 MALA 風險)。
- **MALA 處理**:支持性+矯正誘因;**重度酸中毒(pH <7.1)或腎衰竭蓄積 → 血液透析**(可同時清除 metformin 與矯正酸鹼)。
- **高 AG 酸中毒鑑別(GOLD MARK / MUDPILES)**:乳酸(含 MALA)、酮酸(DKA)、尿毒、甲醇/乙二醇、水楊酸等;本例補碳酸氫鈉仍難矯正且 CKD+metformin → 優先乳酸。
- **各類口服藥主要風險對照**:SU/glinide→低血糖;TZD→水腫、心衰、膀胱癌;SGLT2i→生殖泌尿道感染、euglycemic DKA;DPP-4i→安全性高;acarbose→腸胃道。

### 113-079 ・ 正解 A ・ status: draft

**題幹**  
嗜鉻細胞瘤 (pheochromocytoma) 被視為“偉大的偽裝者(the great masquerader)”，以下哪一項描述最正確?

- (A) 嗜鉻細胞瘤的症狀包括高血壓及陣發性的心悸、冒汗、頭痛等。通常其症狀明顯，但也有某些病人可以無症狀達數年之久。 ← 正解
- (B) 嗜鉻細胞瘤的症狀發作頻率不定，但每次發作時間通常大於兩小時。
- (C) 三環抗憂鬱藥(tricyclic antidepressants)、opiates、metoclopramide等藥會減少嗜鉻細胞瘤的症狀表現，延緩診斷。
- (D) 病人懷疑有嗜鉻細胞瘤需要做電腦斷層或核磁共振等影像學檢查時，為減少刺激，最好不要施打顯影劑。
- (E) 與其他檢驗方式比較，檢測血中的metanephrine 具有高敏感度，但因很容易受到靜脈穿刺 (venipuncture) 等壓力影響，而有較高的偽陽性。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考嗜鉻細胞瘤(pheochromocytoma)被稱為「偉大偽裝者」的**臨床表現、發作型態、誘發藥物、影像與生化診斷**整合觀念。核心:**陣發性高血壓伴頭痛、心悸、冒汗三聯徵**,但表現異質,可長期無症狀或被當其他疾病。

### 選項分析
- **(A) 症狀含高血壓及陣發性心悸/冒汗/頭痛,通常明顯但部分病人可無症狀數年** — **正確 → 本題答案**。此即「great masquerader」精髓:**典型三聯徵(頭痛、心悸、冒汗)+ 高血壓**,但因兒茶酚胺分泌呈陣發,部分腫瘤(尤偶發瘤、家族篩檢發現者)可**長期無症狀**,故易被忽略或誤診。
- **(B) 發作時間通常 >2 小時** — 錯誤。**典型發作多為數分鐘至 1 小時內(常 <15–60 分鐘)**的陣發性「spell」,突發突止;>2 小時並非其特徵。
- **(C) TCA、opiates、metoclopramide 會減少症狀、延緩診斷** — 錯誤。這些藥物恰恰**會誘發/惡化兒茶酚胺釋放與高血壓危象**(metoclopramide、glucagon、TCA 等屬已知誘發劑),不是減少症狀。
- **(D) 影像時為減少刺激最好不要打顯影劑** — 錯誤。現代**非離子性低滲顯影劑不需常規 α-阻斷預處理即可安全施打**;舊觀念已修正,且需顯影劑以利定位/鑑別。臨床定位常用 CT/MRI,功能性影像用 ¹²³I-MIBG 或 ⁶⁸Ga-DOTATATE。
- **(E) 血中 metanephrine 高敏感但易受靜脈穿刺壓力影響有高偽陽性** — 部分正確但敘述不夠精準/非最佳。**血漿游離 metanephrines 敏感度最高(~96–99%),為首選篩檢**;偽陽性主因是**藥物干擾與採血前未平躺休息/體位**,而非單純穿刺;此選項把高偽陽性過度歸因於 venipuncture,作為「最正確」描述不如(A)。

### 答案解析
題問「最正確」。(A)準確描述典型三聯徵且涵蓋可長期無症狀的偽裝特性,完全契合「great masquerader」;其餘各有事實錯誤(發作時長、誘發藥、顯影劑、生化解讀),故選 **(A)**。

### 核心知識點
- **典型表現**:陣發性**頭痛+心悸+冒汗**三聯徵 + 高血壓(可陣發或持續);發作短(數分鐘~1 小時)。**5H**:Hypertension、Headache、Hyperhidrosis、Heart palpitation、Hypermetabolism。
- **生化首選**:**血漿游離 metanephrines 或 24 小時尿 fractionated metanephrines**(敏感度最高);採血前平躺休息、停干擾藥(TCA、acetaminophen、labetalol 等)。
- **誘發危象藥物**:metoclopramide、glucagon、TCA、未先 α-block 即用 β-blocker、麻醉誘導等。
- **術前準備**:先 **α-阻斷(phenoxybenzamine 或 doxazosin)10–14 天**,擴容後再加 β-阻斷控制心搏;**絕不可先 β 後 α**(會致 unopposed α 高血壓危象)。
- **影像/基因**:CT/MRI 定位、MIBG 或 DOTATATE 功能性掃描;約**30–40% 與遺傳症候群相關**(VHL、MEN2、NF1、SDHx),建議基因檢測。

### 本題觀念
本題考尿崩症(DI)的病因分類(中樞性 vs 腎源性 vs 妊娠性)與電解質誘因。中樞性 DI = ADH 分泌不足(下視丘/垂體後葉病變);腎源性 DI = 腎臟對 ADH 反應不良;妊娠性 DI = 胎盤 vasopressinase 分解 ADH。電解質方面,**高血鈣與低血鉀**會干擾腎集尿管對 ADH 的反應而致腎源性 DI。

### 選項分析
- **(A) Sheehan 症候群可造成中樞性 DI** — 正確。產後垂體缺血壞死主要影響前葉,但可累及後葉/漏斗部而致 ADH 不足 → 中樞性 DI(雖較全垂體前葉低下少見)。
- **(B) 鋰鹽與 cisplatin 可造成腎源性 DI** — 正確。**鋰鹽**是腎源性 DI 最經典藥物(干擾集尿管 ADH 訊號、下調 AQP2);**cisplatin** 之腎小管毒性亦可致腎源性 DI。
- **(C) Sarcoidosis 可造成中樞性與腎源性 DI** — 正確。肉芽腫浸潤下視丘/垂體 → 中樞性;高血鈣(sarcoidosis 常見)→ 腎源性,故兩者皆可能。
- **(D) 懷孕第二、三孕期可造成 DI** — 正確。胎盤分泌 **vasopressinase(cysteine aminopeptidase)** 在中後期升高,加速 ADH 代謝 → 妊娠性 DI;對 desmopressin(不被該酶分解)反應佳。
- **(E) 高血鈣與高血鉀可造成腎源性 DI** — **錯誤 → 本題答案**。**腎源性 DI 的電解質誘因是高血鈣(hypercalcemia)與低血鉀(hypokalemia)**,而非高血鉀;高血鉀並非典型誘因。此為「高/低血鉀」混淆陷阱。

### 答案解析
題問「何者不正確」。腎源性 DI 的電解質誘因應為**高血鈣 + 低血鉀**,(E)誤寫成高血鉀,故為錯誤敘述、即本題答案 **(E)**。

### 核心知識點
- **DI 分類**:中樞性(ADH 不足:腫瘤、外傷、術後、Sheehan、肉芽腫、自體免疫)、腎源性(ADH 反應差:**鋰鹽、高血鈣、低血鉀**、遺傳 AVPR2/AQP2)、妊娠性(vasopressinase)。
- **鑑別**:**禁水試驗 + desmopressin**——中樞性給 DDAVP 後尿滲透壓明顯上升;腎源性無反應;妊娠性對 DDAVP 有反應(酶不分解)。新近以 **copeptin** 輔助鑑別。
- **治療**:中樞性/妊娠性用 **desmopressin(DDAVP)**;腎源性處理誘因(停鋰鹽、矯正鈣/鉀)+ **thiazide ± 低鹽飲食 ± amiloride(鋰相關)、NSAID**。
- **記憶**:腎源性 DI 電解質 = **「高鈣低鉀」**(高血鉀不是)。

### 本題觀念
本題考降血糖藥的腫瘤相關禁忌:糖尿病合併**膀胱癌**時,哪一類藥**不可使用**。答案為 **TZD(pioglitazone)**——其與膀胱癌風險增加相關,FDA 與仿單載明**活動性膀胱癌禁用**、有膀胱癌病史者謹慎。

### 選項分析
- **(A) Metformin** — 可用。無膀胱癌禁忌;反而觀察性研究多顯示與較低癌症風險相關。
- **(B) SGLT2 抑制劑** — 可用。與膀胱癌無禁忌關係(早期 dapagliflozin 曾有膀胱癌不平衡的疑慮,但後續資料未確立因果,非禁忌)。

### 113-112 ・ 正解 E ・ status: draft

**題幹**  
低血糖是糖尿病治療過程中，以及非糖尿病患者相關疾病或使用特定藥物治療時常見的併發症，下列何者是發生低血糖的原因? (1) 不當使用胰島素或口服降血糖藥物。 (2) 肢端肥大症。 (3) 胃切除手術後 (Post-gastric bypass surgery)。 (4) 腎上腺機能不足 (Adrenal insufficiency)。 (5) 庫欣氏症候群 (Cushing's syndrome)。

- (A) (1)+(2)+(4)。
- (B) (2)+(3)+(4)。
- (C) (1)+(2)+(5)。
- (D) (1)+(3)+(5)。
- (E) (1)+(3)+(4)。 ← 正解

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考低血糖的病因辨識:逐一判斷 5 個情況是否為**低血糖原因**。關鍵在分辨**會致低血糖**(藥物、餐後傾倒、皮質醇/反調節荷爾蒙不足)與**會致高血糖**(肢端肥大、Cushing,因 GH/皮質醇升高為對抗胰島素)。

### 選項分析(逐條檢核)
- **(1) 不當使用胰島素或口服降糖藥** — 是。胰島素、**磺醯尿素/glinide** 過量或進食不足是低血糖最常見原因。
- **(2) 肢端肥大症(acromegaly)** — 否。**GH 過多 → 拮抗胰島素、升高血糖、致糖尿傾向**,造成的是高血糖/胰島素阻抗,而非低血糖。
- **(3) 胃繞道/胃切除術後(post-gastric bypass)** — 是。**晚期傾倒症候群(late dumping)/餐後高胰島素性低血糖**,食物快速進入小腸→GLP-1/胰島素過度分泌→反應性低血糖。
- **(4) 腎上腺機能不足(adrenal insufficiency)** — 是。**皮質醇為重要反調節荷爾蒙**,不足時糖質新生/升糖反應受損 → 易低血糖(尤空腹/壓力時),為 Addison's 典型表現之一。
- **(5) 庫欣氏症候群(Cushing's syndrome)** — 否。**皮質醇過多 → 升高血糖、致類固醇性糖尿病/胰島素阻抗**,造成高血糖而非低血糖。

### 答案解析
題問哪些是「低血糖的原因」。**(1) 降糖藥不當使用、(3) 胃繞道術後反應性低血糖、(4) 腎上腺機能不足(皮質醇不足)** 皆會致低血糖;**(2) 肢端肥大與 (5) Cushing 因 GH/皮質醇升高反而致高血糖**。故選 **(E) (1)+(3)+(4)**。

### 核心知識點
- **低血糖病因(非糖尿病/反調節相關)**:藥物(**胰島素、SU、酒精**)、**胃繞道後晚期傾倒(餐後高胰島素)**、**內生性高胰島素血症(insulinoma、NIPHS)**、**反調節荷爾蒙不足(腎上腺/垂體低下、皮質醇/GH 缺乏)**、敗血症/肝衰竭、惡病質、非胰島細胞瘤(IGF-2)。
- **方向辨別**:**升高血糖 → GH(肢端肥大)、皮質醇(Cushing)、升糖素、兒茶酚胺**;**降低血糖 → 胰島素、皮質醇「不足」**。
- **評估(低血糖三聯徵成立時)**:同步測**胰島素、C-peptide、proinsulin、β-OHB、磺醯尿素篩檢、皮質醇**;C-peptide 區分內生 vs 外源胰島素;皮質醇低 → 查腎上腺/垂體軸。
