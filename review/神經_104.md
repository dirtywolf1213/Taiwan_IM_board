### 104-179 ・ 正解 B ・ status: draft

**題幹**  
20歲女性因左手無力就診，神經學檢查顯示左側的手指無法完全張開與閉合，掌骨間肌（interossei muscle）無力萎縮，是那一條神經受損？

- (A) 正中神經（median nerve）
- (B) 尺神經（ulnar nerve） ← 正解
- (C) 橈神經（radial nerve）
- (D) 肌表皮神經（musculocutaneous nerve）
- (E) 長胸神經（long thoracic nerve）

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**周邊神經定位**。關鍵理學徵候:**掌骨間肌(interossei)無力萎縮 + 手指無法完全張開與閉合**。骨間肌(背側+掌側)與第 3、4 蚓狀肌、小指外展肌等手部小肌肉幾乎都由**尺神經(ulnar nerve, C8–T1)**支配;這些肌肉負責手指的外展 / 內收(張開閉合)。骨間肌萎縮造成掌骨間凹陷,是尺神經病變的典型表現。

### 選項分析
- **(A) 正中神經(median nerve)** — 錯誤。正中神經支配前臂大部分屈肌、**大魚際(LOAF:lumbricals 1-2、opponens、abductor pollicis brevis、flexor pollicis brevis)**,病變表現為大魚際萎縮、拇指對掌困難與正中神經感覺區麻木,非骨間肌。
- **(B) 尺神經(ulnar nerve)** — **正確 → 本題答案**。支配**所有骨間肌、第 3/4 蚓狀肌、小魚際肌、拇內收肌**;病變致骨間肌萎縮、爪形手(ulnar claw)、Froment's sign、手指外展內收無力,完全符合題述。
- **(C) 橈神經(radial nerve)** — 錯誤。主管**伸腕伸指**,病變為**垂腕(wrist drop)**,不影響手內在小肌肉的外展內收。
- **(D) 肌皮神經(musculocutaneous nerve)** — 錯誤。支配上臂屈肌(biceps、brachialis),病變致屈肘無力與前臂外側感覺異常,與手部無關。
- **(E) 長胸神經(long thoracic nerve)** — 錯誤。支配前鋸肌,病變造成**翼狀肩胛(scapular winging)**,與手指完全無關。

### 答案解析
骨間肌萎縮 + 手指外展內收(張開閉合)障礙是**尺神經**支配區的特徵,故選 **(B)**。

### 核心知識點
- **尺神經(C8–T1)**:支配**所有骨間肌 + 第 3/4 蚓狀肌 + 小魚際 + 拇內收肌 + FCU + FDP(4/5 指)**。
  - 徵候:**爪形手(claw hand)**(第 4/5 指)、**Froment's sign**(夾紙用拇長屈肌代償)、骨間肌萎縮、第一背側骨間肌無力。
  - **尺神經悖論(ulnar paradox)**:**肘部**(較高位)損傷的爪形手反而比**腕部**輕。
- **正中神經**:LOAF 肌、猿手(ape hand);腕隧道症候群最常見。
- **橈神經**:wrist drop;Saturday night palsy。
- 定位口訣:**伸=橈、對掌大魚際=正中、手內在小肌(外展內收)=尺**。

2. Preston DC, Shapiro BE. *Electromyography and Neuromuscular Disorders*. 3rd ed. Elsevier; 2013(Ulnar Neuropathy).

### 104-180 ・ 正解 E ・ status: draft

**題幹**  
25歲女性有紅斑性狼瘡（systemic lupus erythematosus）合併腎炎有3年，使用類固醇、免疫抑制劑（cyclophosphamide），因意識不清與癲癇至急診，理學檢查血壓200/110 mmHg、體溫37°C，脊髓液檢查顯示無白血球、蛋白濃度稍增加，腦部磁振照影顯示兩側枕葉有不正常訊號，最可能的診斷為何？

- (A) 腦膜腦炎（meningoencephalitis）
- (B) 尿毒腦病變（uremic encephalopathy）
- (C) 急性去髓鞘腦炎（acute demyelinating encephalitis）
- (D) 急性腦梗塞（acute cerebral infarction）
- (E) 可逆性後腦病變症候群（posterior reversible encephalopathy syndrome） ← 正解

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題情境高度典型:**SLE + 高血壓危象(200/110)+ 免疫抑制劑(cyclophosphamide / 類固醇)+ 癲癇與意識不清 + 兩側枕葉(後循環)MRI 異常訊號 + CSF 幾乎正常(無白血球、蛋白略升)**。這組合指向 **可逆性後腦病變症候群(PRES, posterior reversible encephalopathy syndrome)**——血壓急遽升高 / 內皮損傷 / 免疫抑制劑使後循環(頂枕葉)血腦障壁破壞而水腫。

### 選項分析
- **(A) 腦膜腦炎(meningoencephalitis)** — 錯誤。感染性腦膜腦炎 CSF 應有**白血球增多 / 明顯蛋白上升 / 糖變化**;本例 CSF 近乎正常,不支持。
- **(B) 尿毒腦病變(uremic encephalopathy)** — 錯誤。雖有狼瘡腎炎,但尿毒腦病變影像通常**非局限枕葉**,且本題未提氮血症數值;典型表現為瀰漫腦病而非後部對稱水腫。
- **(C) 急性去髓鞘腦炎(ADEM)** — 錯誤。ADEM 多為感染 / 疫苗後、**多發白質病灶(不限枕葉)**,且與急性高血壓無關。
- **(D) 急性腦梗塞(acute cerebral infarction)** — 錯誤。雙側對稱枕葉水腫不符單一血管分布;PRES 為**血管源性水腫且可逆**,與梗塞(細胞毒性、不可逆、限於血管領域)不同。
- **(E) 可逆性後腦病變症候群(PRES)** — **正確 → 本題答案**。**高血壓 + 免疫抑制劑 + 子癇 / 自體免疫**為三大誘因;影像為**頂枕葉對稱血管源性水腫(T2/FLAIR 高訊號)**,控制血壓 / 停藥後可逆。

### 答案解析
高血壓危象 + 免疫抑制劑 + 雙側枕葉病灶 + CSF 近正常,典型為 PRES,故選 **(E)**。

### 核心知識點
- **PRES 誘因**:**血壓急遽升高、子癇前症/子癇、免疫抑制劑(cyclosporine、tacrolimus、cyclophosphamide)、自體免疫(SLE)、腎衰竭**。
- **影像**:T2/FLAIR **頂枕葉對稱性血管源性水腫**(DWI 多不受限,可與梗塞區分);非典型可侵犯額葉、小腦、腦幹。
- **臨床**:頭痛、視覺障礙、癲癇、意識改變,**及時降壓 / 停致病藥多可逆**。
- **降壓原則**:急性期數小時內降約 **20–25%**,避免過快;首選靜脈 **labetalol / nicardipine**。

### 參考資料

PMID: [26184985](https://pubmed.ncbi.nlm.nih.gov/26184985/). DOI: [10.1016/S1474-4422(15)00111-8](https://doi.org/10.1016/S1474-4422%2815%2900111-8).
2. Hinchey J, Chaves C, Appignani B, et al. A reversible posterior leukoencephalopathy syndrome. *N Engl J Med*. 1996;334(8):494-500. PMID: [8559202](https://pubmed.ncbi.nlm.nih.gov/8559202/). DOI: [10.1056/NEJM199602223340803](https://doi.org/10.1056/NEJM199602223340803).

</details>

- (B) (2)+(4)
- (C) (1)+(4)
- (D) (2)+(3)
- (E) (1)+(2)+(3)+(4)

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**重症肌無力(MG)病人應避免的藥物**。許多抗生素會**干擾神經肌肉接合處傳遞**(抑制突觸前 ACh 釋放或阻斷突觸後受器),可誘發 MG 惡化甚至肌無力危象。其中**fluoroquinolones 與 macrolides 是公認應避免**的兩類(FDA 對 fluoroquinolones 有 MG 黑框警告)。題目問「較不適合」的組合即 (1)+(3)。

### 選項分析
- 藥類 **(1) Quinolones** — **應避免**。Fluoroquinolones 可顯著惡化 MG,FDA 黑框警告;與 macrolides 同列最該避免。
- 藥類 **(2) Beta-lactam + beta-lactamase inhibitor** — 相對安全。β-lactam 類(含 augmentin)為 MG 病人常用首選。
- 藥類 **(3) Macrolides** — **應避免**。Azithromycin / erythromycin / clarithromycin 可在數分鐘至小時內誘發 MG 惡化。
- 藥類 **(4) Cephalosporins** — 相對安全。與 penicillin 同屬 β-lactam,為 MG 感染的首選治療之一。
- 故較不適合者為 (1)+(3)。
- **(A) (1)+(3)** — **正確 → 本題答案**。Quinolones + macrolides 兩類皆會惡化 MG。
- **(B) (2)+(4)** — 錯誤。此兩類(含酶抑制劑的β-lactam、cephalosporin)恰為相對安全選擇。
- **(C) (1)+(4)** — 錯誤,cephalosporin 安全。
- **(D) (2)+(3)** — 錯誤,β-lactam/酶抑制劑安全。
- **(E) 四者皆避免** — 錯誤,β-lactam 與 cephalosporin 並非禁忌。

題幹問「較不適合使用」。Fluoroquinolones 與 macrolides 是 MG 公認應避免的抗生素,即 (1)+(3),故選 **(A)**。

### 核心知識點
- **MG 應避免 / 慎用藥物**:
  - 抗生素:**aminoglycosides、fluoroquinolones、macrolides、telithromycin(禁用)、polymyxin / colistin**。
  - 其他:**IV magnesium(強烈惡化)、β-blockers、procainamide / quinidine、神經肌肉阻斷劑(D-tubocurarine 等)、penicillamine(誘發 MG)、checkpoint inhibitors、statins**。
- **相對安全抗生素**:**penicillins、cephalosporins、carbapenems、vancomycin、TMP-SMX**。
- **肌無力危象**處置:**呼吸支持 + 血漿置換(plasmapheresis)或 IVIG**;類固醇起始可能短暫惡化需留意。
- 口訣:抗生素惡化 MG → **"AFM" = Aminoglycosides、Fluoroquinolones、Macrolides**。

### 104-183 ・ 正解 B ・ status: draft

**題幹**  
相較於肌無力（myasthenia gravis），藍伯-伊頓肌無力症（Lambert-Eaton myasthenic syndrome）較常出現那些症狀？(1) 眼皮下垂、複視；(2) 肌腱反射下降；(3) 口齒不清及吞嚥困難；(4) 自律神經失調症狀，如口乾、便秘、小便困難。

- (A) (1)+(3)
- (B) (2)+(4) ← 正解
- (C) (1)+(4)
- (D) (2)+(3)
- (E) (1)+(2)+(3)+(4)

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考 **Lambert-Eaton 肌無力症候群(LEMS)與重症肌無力(MG)的鑑別**。LEMS 為**突觸前**病變(抗 P/Q 型電壓閘控鈣離子通道抗體 → ACh 釋放減少),典型三聯:**近端肢體無力(下肢為主)、肌腱反射下降 / 消失、自律神經失調(口乾、便秘、勃起障礙)**,且運動後肌力短暫改善(post-exercise facilitation)。題目問 LEMS 相較 MG **較常**出現者,即 (2)+(4)。

### 選項分析
- 症狀 **(1) 眼皮下垂、複視** — 較屬 MG。MG **眼外肌侵犯為早期且常見**;LEMS 眼徵相對少且輕。
- 症狀 **(2) 肌腱反射下降** — **LEMS 特徵**。反射減弱 / 消失,運動後可暫時恢復(facilitation);MG 反射正常。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題考**warfarin 相關顱內出血(ICH)的緊急逆轉**:機械瓣 + 心房顫動長期口服抗凝、INR 5.5、發生腦出血。重點在「**最快**恢復凝血功能」的組合。藥理上**最快**逆轉者為**第四代凝血酶原複合體濃縮劑(4-factor prothrombin complex concentrate, PCC)**(數分鐘內補足 II、VII、IX、X,且不需大量輸液),搭配**維生素 K**(維持後續因子合成)。FFP 可逆轉但較慢、量大、有容量負荷與配對等待之缺點。

> 本題官方判定為**送分(A、B、C 皆給分)**。推測因選項組合與「最快」之題意間存有爭議——含 PCC 的 (C)〔(1)+(2)+(4)〕在藥理上才是「最快」,但官方接受 A、B、C。以下說明各成分的角色與正確觀念。

### 選項分析(逐成分判斷角色)
- **(1) Vitamin K** — 必給。**靜脈 5–10 mg**,作用慢(數小時起、約 24 小時完全),用於**維持**逆轉、避免 PCC/FFP 作用消退後 INR 反彈;單用不夠快。
- **(2) Fresh frozen plasma(FFP)** — 可逆轉但**較慢**(需解凍、配對、大量輸注 ~15 mL/kg),且有**容量過負荷/TRALI**風險;在無 PCC 時為替代。
- **(3) Platelets** — 對 warfarin 凝血因子缺乏**無助**(除非合併血小板低下/抗血小板藥);屬陷阱成分。
- **(4) Prothrombin complex concentrate(PCC)** — **逆轉最快**(數分鐘補齊維生素 K 依賴因子,小體積),為現行**首選**,須與 vitamin K 併用。
- **(5) Whole blood** — 非標準逆轉手段,因子濃度低、量大,不用於緊急逆轉(陷阱)。
- 含 PCC 的 **(C) (1)+(2)+(4)** 在「最快」題意下最合理;**(A)(B)** 不含 PCC,逆轉較慢,但官方一併送分。**→ 本題答案：A、B、C(送分)**。

### 答案解析
本題官方**送分(A、B、C 皆可)**。就藥理而言含 **PCC + vitamin K** 的組合恢復凝血最快;platelets 與 whole blood 對 warfarin 逆轉無實質幫助。故記住正確觀念:**PCC + IV vitamin K 為首選**,FFP 為次選。

### 核心知識點
- **Warfarin 相關大出血/ICH 逆轉(首選)**:**4-factor PCC(依 INR 與體重給,約 25–50 IU/kg)＋ IV vitamin K 10 mg**。PCC 起效最快、體積小,優於 FFP。
- **FFP**:無 PCC 時用,15 mL/kg;缺點為慢、量大、容量負荷。
- **無效/不該用**:platelets(除非血小板低或合併抗血小板)、whole blood。
- **機械瓣病人**:逆轉後血栓風險高,**待出血穩定後盡早重啟抗凝**(時機個別化,常 1–2 週)。
- **DOAC 對照**:dabigatran→idarucizumab;Xa 抑制劑→andexanet alfa 或 PCC。
