本題考**皮膚黑色素細胞癌(cutaneous melanoma)的分期治療與標靶/免疫藥物分類**。重點:早期(I–II 期)以**廣泛局部切除(wide local excision)**為主、必要時加哨兵淋巴結切片;轉移性疾病的治療已由化療轉為以**免疫檢查點抑制劑與 BRAF/MEK 標靶**為主流。需逐一判讀五個敘述真偽。

### 選項分析

- (A) 鱗狀細胞癌 (Squamous cell carcinoma)；因為日曬引起P53基因突變造成。

- (C) 基底細胞癌 (Basal cell carcinoma)；由於Hedgehog訊息傳遞路徑活化所造成。 ← 正解

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題以一位 **86 歲男性**的皮膚病灶圖,考**基底細胞癌(BCC)及其致病機轉**。BCC 是最常見的皮膚癌,好發於**慢性日曬部位(頭頸、鼻部)**的老年人,典型為**珍珠樣丘疹、毛細血管擴張、捲邊潰瘍(rodent ulcer)**。其分子核心為 **Hedgehog(Hh)訊息傳遞路徑的異常活化**——多因 **PTCH1 功能喪失(>85%)** 或 **SMO 活化突變(約 10%)**,使 GLI 轉錄因子持續活化而致癌。

### 選項分析
- **(A) 鱗狀細胞癌;日曬→P53 突變** — 錯誤。SCC 機轉(UV→p53 突變)敘述本身正確,但 86 歲、慢性日曬部位的珍珠樣潰瘍病灶最符合 BCC,非 SCC。
- **(B) 血管肉瘤;SMO 表現偏低** — 錯誤。血管肉瘤好發頭皮、預後差,且機轉與 SMO 無關;SMO 在 BCC 是**被活化**而非偏低,敘述矛盾。
- **(C) 基底細胞癌;Hedgehog 路徑活化** — **正確 → 本題答案**。臨床(老年、日曬部位、珍珠樣捲邊潰瘍)與機轉(Hh/PTCH1-SMO-GLI 活化)完全相符;靶向 SMO 的 vismodegib 即用於進展期 BCC。
- **(D) 黑色素細胞癌;BRAF 突變** — 錯誤。BRAF V600 突變確為黑色素瘤機轉,但圖示與情境指向 BCC,非色素病灶。
- **(E) 梅克爾細胞癌;MCPyV 病毒** — 錯誤。Merkel cell carcinoma 與 **Merkel cell polyomavirus** 有關屬正確配對,但其為快速增大的紅紫色結節,非典型 BCC 表現,屬干擾項。

### 答案解析
題問「最可能診斷與致病機轉何者正確」。老年慢性日曬部位的珍珠樣潰瘍 + Hedgehog 路徑活化,唯 (C) 診斷與機轉皆正確,故選 C。

### 核心知識點
- **BCC 致病核心**:**Hedgehog 路徑活化**——**PTCH1 失活(>85%)** 解除對 **SMO** 的抑制 → **SMO** 活化 → **GLI** 轉錄因子致癌;**UV 暴露**為主要環境因子,**Gorlin 症候群(痣樣基底細胞癌症候群)** 為 PTCH1 胚系突變。
- **臨床**:珍珠樣丘疹、毛細血管擴張、捲邊潰瘍;好發頭頸,**極少轉移**但局部侵犯。
- **治療**:手術切除(高風險用 Mohs);進展/轉移性用 **SMO 抑制劑 vismodegib / sonidegib**,或抗 PD-1(cemiplimab)。

<details><summary><b>目前詳解</b>(點開)</summary>

### 本題觀念
本題以 **40 歲女性**的皮膚病灶圖,考**貝西氏症(Behçet syndrome)**的診斷與關鍵臨床特徵。Behçet 是一種血管炎,以**反覆口腔潰瘍(必要條件)**為核心,合併**生殖器潰瘍、眼部病變(葡萄膜炎)、皮膚病灶(結節性紅斑、假性毛囊炎、針扎反應 pathergy)**。地中海至東亞「絲路」沿線盛行,常見於年輕成人。

### 選項分析
- **(A) 貝西氏症;口腔潰瘍、結節性紅斑** — **正確 → 本題答案**。反覆口腔潰瘍為診斷必要條件,**結節性紅斑(erythema nodosum)** 是其最具代表性的皮膚表現之一,符合年輕女性的病灶。
- **(B) 皮肌炎;Gottron 氏徵候** — 錯誤。Gottron sign(指關節伸側紫紅丘疹)是皮肌炎的指標性表現,但與本題口腔/結節性紅斑情境不符,屬不同疾病的特徵配對陷阱。
- **(C) 類風溼性關節炎;類風濕性結節** — 錯誤。類風濕結節見於關節伸側皮下,主病為對稱性多關節炎,非以口腔潰瘍/結節性紅斑表現。
- **(D) 全身性硬皮症;手指硬化、手指潰瘍** — 錯誤。sclerodactyly 與指端潰瘍是硬皮症特徵,屬另一疾病的皮膚表現混淆。
- **(E) 感染性心內膜炎;Janeway lesion** — 錯誤。Janeway lesion(掌蹠無痛性出血斑)為 IE 周邊徵象,與本題不符。

### 答案解析
題問「最可能診斷與最重要臨床特徵」。年輕女性合併口腔潰瘍與結節性紅斑最符合 Behçet,故選 (A)。

- **特色機轉**:可侵犯**任何大小、動靜脈**血管(unique 之處);**靜脈血栓、肺動脈瘤**為重症表現。
- **治療**:黏膜皮膚/關節用 **colchicine、局部/全身類固醇**;重症(眼、神經、大血管)用 **azathioprine、ciclosporin、TNF-α 抑制劑(infliximab/adalimumab)、apremilast(口腔潰瘍)**。

### 本題觀念
本題以 **56 歲女性**的皮膚病灶圖,考**皮肌炎(dermatomyositis, DM)的特徵性皮膚表現**。DM 是一種自體免疫性發炎性肌病,皮膚徵象常先於或伴隨近端對稱性肌無力出現,具高度診斷價值的皮膚徵象包括 **Gottron 丘疹/徵候、heliotrope rash(眼瞼紫紅色斑合併水腫)、shawl sign / V sign(光照部位紫紅斑)、技工手(mechanic's hands)、指甲周圍微血管異常**。成人 DM 須警覺**合併內臟惡性腫瘤**。

### 選項分析
- **(A) 圓盤型紅斑性狼瘡(DLE)** — 錯誤。DLE 為界限清楚、有黏附性鱗屑與毛囊角栓、會留疤與色素變化的圓盤狀斑塊,多在頭面;與 DM 的紫紅色光敏分布不同。
- **(B) 皮肌炎(dermatomyositis)** — **正確 → 本題答案**。圖示符合 DM 特徵性皮膚徵象(如 heliotrope、Gottron、shawl/V sign 等紫紅色光敏皮疹),為本題診斷。
- **(C) 血管炎(vasculitis)** — 錯誤。皮膚血管炎多為**可觸及性紫斑(palpable purpura)**,好發下肢,與 DM 的紫紅色斑塊分布不同。
- **(D) 硬皮症(scleroderma)** — 錯誤。硬皮症以皮膚硬化、雷諾現象、sclerodactyly、毛細血管擴張為主,無 DM 的特徵性紫紅皮疹。
- **(E) 類澱粉沉著症(amyloidosis)** — 錯誤。皮膚類澱粉常見眼周「pinch purpura」、蠟樣丘疹、巨舌,與本題分布不符。

### 答案解析
題問「最可能診斷」。中年女性具特徵性紫紅色光敏皮疹(heliotrope / Gottron / shawl sign)最符合皮肌炎,故選 (B)。

### 本題觀念
本題以 **44 歲男性、有心臟病變**且皮膚出現病灶,考**系統性類澱粉沉著症(systemic amyloidosis)的皮膚表現**。關鍵連結是「**心臟侵犯 + 特徵性皮膚徵象**」:AL(免疫球蛋白輕鏈)型類澱粉常同時侵犯心臟(限制型心肌病/心衰)與皮膚,典型皮膚表現為**輕微外傷或揉捏後出現的紫斑(pinch purpura)、眼周紫斑('raccoon eyes')、蠟樣半透明丘疹/斑塊與巨舌(macroglossia)**,因類澱粉沉積使血管脆弱所致。

### 選項分析
- **(A) 圓盤型紅斑性狼瘡(DLE)** — 錯誤。DLE 為圓盤狀鱗屑斑塊、毛囊角栓、留疤,不會合併心臟類澱粉沉積這類系統表現。
- **(B) 皮肌炎(dermatomyositis)** — 錯誤。DM 以紫紅色光敏皮疹與近端肌無力為主,雖可有心肌炎,但皮膚徵象與 pinch purpura/眼周紫斑不同。
- **(C) 血管炎(vasculitis)** — 錯誤。血管炎以可觸及性紫斑為主,且本題「心臟病變+脆弱性紫斑/蠟樣丘疹」更指向類澱粉沉著。
- **(D) 硬皮症(scleroderma)** — 錯誤。硬皮症以皮膚硬化、雷諾、肺/腎/食道侵犯為主,無類澱粉的眼周紫斑與巨舌。
- **(E) 類澱粉沉著症(amyloidosis)** — **正確 → 本題答案**。中年男性合併**心臟病變**與特徵性皮膚紫斑/蠟樣丘疹,最符合系統性(AL)類澱粉沉著,故為本題診斷。

### 答案解析
題問「最可能診斷」。心臟侵犯加上脆弱性紫斑、眼周紫斑、蠟樣丘疹等皮膚徵象,最符合系統性類澱粉沉著症,故選 (E)。

### 核心知識點
- **皮膚類澱粉徵象**:**Pinch purpura(揉捏/外傷後紫斑)**、**眼周紫斑(raccoon eyes)**、蠟樣半透明丘疹斑塊、**巨舌(macroglossia)**——皆因類澱粉沉積使血管脆弱。
- **系統 AL 型**:常侵犯**心臟(限制型心肌病、低電壓 ECG、NT-proBNP/troponin 升高)、腎(腎病症候群)、神經、肝**;與**多發性骨髓瘤/漿細胞病變**相關。
- **診斷**:組織切片 **Congo red 染色 → 偏振光下蘋果綠雙折射**;以質譜分型(AL vs ATTR)。心臟 ATTR 可用骨骼閃爍掃描(PYP/DPD)。
- **治療**:AL 型以**化療/抗漿細胞治療(如 daratumumab-based、自體幹細胞移植)**;ATTR 用 tafamidis 等穩定劑。
